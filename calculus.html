נושא 3 מתוך 6
▼
מבחן הנגזרת הראשונה
f′ עוברת (+)→(−): מקסימום
f′ עוברת (−)→(+): מינימום
מבחן הנגזרת השנייה
f′(x₀)=0, f″(x₀)<0 → מקסימום
f′(x₀)=0, f″(x₀)>0 → מינימום
למה מבחן הנגזרת השנייה עובד?
f″(x₀) < 0 אומר שf′ יורדת סביב x₀ — כלומר f′ עוברת מ-(+) ל-(−) = מקסימום.
f″(x₀) > 0 אומר שf′ עולה סביב x₀ — כלומר f′ עוברת מ-(−) ל-(+) = מינימום.
⚠️ כשf″(x₀) = 0: המבחן לא מכריע — חייבים לבדוק ידנית עם טבלת סימנים של f′.

f″(x) = 6x  ← כדי לסווג כל נקודה נשתמש במבחן הנגזרת השנייה

x = 1: f″(1) = 6·1 = 6 > 0 → מינימום מקומי  |  f(1) = 1³ − 3·1 = 1−3 = −2
✓ מקסימום מקומי: (−1, 2)  |  מינימום מקומי: (1, −2)

קריאת גרף f′ ו-f″
נושא 5 מתוך 6
▼
מגרף f′ נלמד על f:
• f′ > 0 (גרף f′ מעל ציר x) → f עולה
• f′ < 0 (גרף f′ מתחת לציר x) → f יורדת
• f′(x₀) = 0 (גרף f′ חוצה את ציר x) → נקודה קריטית של f
• f′ עולה → f קמורה (∪)  |  f′ יורדת → f קעורה (∩)
• f′ מקסימום/מינימום → נקודת פיתול של f
9
נתון הגרף של f′(x):
ענה: (1) היכן f עולה/יורדת? (2) מה טיב נקודות x=a, x=b? (3) היכן f קמורה/קעורה?
(1) f′>0 ל-x<a ול-x>b → f עולה בתחומים אלה
f′<0 ל-a<x<b → f יורדת בתחום זה
(2) x=a: f′ עוברת (+)→(−) → מקסימום מקומי של f
x=b: f′ עוברת (−)→(+) → מינימום מקומי של f
(3) f′ יורדת בכל הגרף המוצג → f″<0 → f קעורה (∩) לכל אורך הגרף
אלא אם כן f′ מגיעה למינימום — שם יש נקודת פיתול של f
✓ עלייה: x<a ו-x>b  |  מקסימום ב-a  |  מינימום ב-b  |  קעורה בין a ל-b
10
נתון גרף f″(x). מה אפשר לדעת על f?
f″<0 ל-x<x₀ → f קעורה (∩) לשמאל של x₀
f″>0 ל-x>x₀ → f קמורה (∪) לימין של x₀
f″(x₀)=0 עם שינוי סימן → נקודת פיתול של f ב-x₀
f″ עולה לאורך כל הגרף → f′ קמורה (∪)
✓ x₀ היא נקודת פיתול של f — מעבר מקעורה לקמורה

קריאת גרף f + ניתוח

נתון גרף הפונקציה f(x) הרציפה. הגרף עולה ל-x<0, יורד ל-0<x<a, ויש נקודת פיתול ב-x=x₁ שבין 0 ל-a. הגרף עובר דרך הנקודה (b, 0).

א. (1) ציין תחומי עלייה וירידה של f.   (2) כמה נקודות פיתול יש לפונקציה?
(1) עלייה: x < 0  |  ירידה: 0 < x < a
(2) יש נקודת פיתול אחת ב-x = x₁ (שינוי קמירות מ-∪ ל-∩ או הפוך)
✓ עלייה: x<0  |  ירידה: 0<x<a  |  נקודת פיתול אחת
ג. מצא את a (נתון שf(a)=0 ו-a>0).
מהגרף: a = 2
✓ a = 2
ה. חשב את השטח הכלוא בין גרף f לציר x בתחום [0, a].
f יורדת מ-f(0) ל-f(a)=0 בתחום [0,2]
שטח = |∫₀ᵃ f(x)dx|
לפי נתוני השאלה המקורית: שטח = 5/24
✓ שטח = 5/24
בגרות תשפ"ב נבצרים
ניתוח פונקציה רציונלית

נתונה הפונקציה: f(x) = (18 − 36x) / (x² − x − 6)²

א. מצא את הנקודות b, a כך שb<a וf(a)=f(b)=0 (הגרף חוצה ציר x).
f(x)=0 כאשר המונה = 0: 18 − 36x = 0 → x = 1/2
המכנה: x²−x−6 = (x−3)(x+2). אסימפטוטות אנכיות: x=3, x=−2
הגרף חוצה ציר x רק ב-x = 1/2. כלומר a = 1/2
b ו-c הן נקודות האסימפטוטה: b = −2, c = 3
✓ a = 1/2, b = −2, c = 3
ד(1). הראה כי בתחום b < x < c מתקיים: f′(x)·(f(x))² ≤ 0.
(f(x))² ≥ 0 תמיד (ריבוע של מספר ממשי)
בתחום b<x<c (כלומר −2<x<3): נגזור את f ונמצא את סימנה
המונה 18−36x: שלילי עבור x>1/2, חיובי עבור x<1/2
ניתן להראות שf′(x) ≤ 0 בכל התחום −2<x<3 (f יורדת שם)
לכן f′(x)·(f(x))² ≤ 0 כי f′≤0 ו-(f)²≥0
✓ מכיוון ש-f′≤0 ו-(f(x))²≥0, מכפלתם ≤ 0
בגרות תשפ"ג מועד א
ניתוח g(x) = x²·f(x)

נתון: f פונקציה גזירה. ידוע: f(a)=0, a<0, f עולה ב-x=a (מקסימום מקומי של f). בנוסף f′(0)=0, f″(0)=0. הגדר g(x) = x²·f(x).

א. מצא את g′(x) וקבע אם x=0 נקודת קיצון של g.
g′(x) = 2x·f(x) + x²·f′(x)
g′(0) = 2·0·f(0) + 0²·f′(0) = 0
לבדיקת סוג הנקודה: g″(x) = 2f(x) + 2x·f′(x) + 2x·f′(x) + x²·f″(x) = 2f(x) + 4x·f′(x) + x²·f″(x)
g″(0) = 2f(0) + 0 + 0 = 2f(0)
הסימן של g″(0) תלוי בסימן f(0): אם f(0)>0 → מינימום; אם f(0)<0 → מקסימום
✓ g′(0)=0. סוג הקיצון תלוי בסימן f(0)
ב. מצא את g′(a) (כאשר f(a)=0).

נתון גרף הפונקציה f′(x). מהגרף ניתן לראות: f′(x) = 0 ב-x=b וב-x=c (b<c). f′ חיובית מחוץ לתחום (b,c) ושלילית בתוכו.

א. ציין: (1) תחומי עלייה/ירידה של f. (2) תחומי קמירות/קעורות.
(1) f′>0 ל-x<b ול-x>c → f עולה בתחומים אלה
f′<0 ל-b<x<c → f יורדת
(2) תחומי קמירות/קעורות נקבעים לפי f″ = (f′)′:
היכן f′ עולה → f קמורה (∪)
היכן f′ יורדת → f קעורה (∩)
✓ עלייה: x<b ו-x>c  |  ירידה: b<x<c
ג. מהגרף: b = −2, c = 3, ונקודת ה-x שבה f′ מקסימלית היא x = a = 1/2.
מהגרף של f′ קוראים ישירות: a = 1/2, b = −2, c = 3
x=a=1/2 הוא נקודת מינימום של f′ (f′ מינימלית שם)
→ f″(1/2) = 0 ושינוי סימן → נקודת פיתול של f ב-x=1/2
✓ a=1/2, b=−2, c=3  |  נקודת פיתול של f ב-x=1/2
