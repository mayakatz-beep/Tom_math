מה זו בעיית קיצון?

  • ניתנת מציאות (שטח, היקף, נפח, מרחק, רווח…)
  • יש לנו מגבלה (constraint) — קשר בין המשתנים
  • רוצים למצוא מקסימום או מינימום של כמות כלשהי

5 הצעדים לפתרון כל בעיית קיצון:

  • הגדר משתנה: בחר משתנה אחד (בדרך כלל x או t) שמתאר את הגודל "הנבחר" בבעיה. למה? כי כל שאר הגדלים ייבטאו דרכו.
  • בטא פונקציית מטרה: השתמש במגבלה (constraint) כדי לבטא את הכמות שממקסמים/ממזערים כפונקציה של משתנה אחד בלבד. למה? כי נגזרת פועלת על פונקציה של משתנה יחיד.
  • קבע תחום: מה הערכים הפיזיים האפשריים? (x>0, 0<x<L, וכו׳). למה? כי קיצון יכול להיות בתחום או בקצוות — צריך לדעת איפה לחפש.
  • גזור ומצא נקודות קריטיות: f'(x)=0 → מועמדים לקיצון. אמת ע"י f''(x₀) או טבלת סימנים של f'. למה? כי f'=0 הכרחי אך לא מספיק — יכולה להיות נקודת פיתול.
  • השווה קצוות ומסקנה: חשב גם את ערכי הקצוות (אם הם מותרים) והשווה. כתוב פסוק מסקנה מלא. למה? כי הקיצון הגלובלי יכול להיות דווקא בקצה.
💡 כלל הזהב: בעיית קיצון מסתיימת ב-פסוק — "הערך המקסימלי/מינימלי של ... הוא ... ומושג ב-x = ..."
מצא קיצון
f'(x) = 0
ואז בדוק סימן f' (+ → − = מקסימום)
בדיקת מינימום
f''(x₀) > 0 ← מינימום
f''(x₀) < 0 ← מקסימום
שטח מלבן
S = a · b
היקף P = 2(a + b)
שטח משולש
S = ½ · b · h
S = ½|x_A·y_B − x_B·y_A|
1
מצא שני מספרים חיוביים שסכומם 20 וכפלם מקסימלי.
שלב 1: נסמן את המספר הראשון x. אז המספר השני = 20 − x (כי סכומם 20).
שלב 2: פונקציית המטרה (הכפל): P(x) = x · (20−x) = 20x − x²  ← פרוסה, כי קל לגזור פולינום
שלב 3: תחום: שני המספרים חיוביים → x > 0 וגם 20−x > 0 → 0 < x < 20
שלב 4: P'(x) = 20 − 2x = 0 → 2x = 20 → x = 10
אימות: P''(x) = −2 < 0 תמיד → פרבולה פתוחה למטה → x=10 הוא מקסימום גלובלי ✓
המספרים: 10 ו-10. הכפל המקסימלי: P(10) = 10·10 = 100
2
יש לגדר שדה מלבני עם 60 מ' של גדר. שלושה מהצלעות ייגדרו (הצלע הרביעית לאורך קיר). מה הממדים שנותנים שטח מקסימלי?
שלב 1: נסמן רוחב השדה x. שתי צלעות רוחב משתמשות בגדר, לכן האורך (צלע אחת): y = 60 − 2x
שלב 2: שטח: S(x) = x · y = x·(60−2x) = 60x − 2x²  ← פונקציית המטרה
שלב 3: x > 0 (רוחב חיובי), ו-60−2x > 0 → x < 30. תחום: 0 < x < 30
שלב 4: S'(x) = 60 − 4x = 0 → 4x = 60 → x = 15
אימות: S''(x) = −4 < 0 → מקסימום ✓  |  אורך: y = 60 − 2·15 = 30 מ׳
שטח: S(15) = 15 · 30 = 450 מ"ר
ממדי השדה האופטימלי: 15 מ׳ (רוחב) × 30 מ׳ (אורך). שטח מקסימלי: 450 מ"ר

מלבן שצלעותיו מקביליות לצירים — הרעיון

  • נקודה A = (t, f(t)) על הגרף — קצה עליון-ימני של המלבן
  • בגלל סימטריה (או ציר x): קצוות המלבן נגזרים מ-t
  • שטח/היקף מתבטאים ב-t — גוזרים ומאפסים
💡 טיפ: בחצי-מעגל f(x) = √(R²−x²), נקודה A בריביע I עם קואורדינטה x=t נותנת: גובה = √(R²−t²), רוחב = 2t (אם סימטרי), או t בלבד (אם ב-[0,R]).
3
מלבן ABCD משוחל בתוך הפרבולה y = 4 − x², כאשר צלע CD על ציר ה-x. נסמן את הקואורדינטה x של A ב-t (t > 0). הבע את שטח המלבן בt ומצא את t שממקסם את השטח.
הגדרת נקודות: A = (t, 4−t²), B = (−t, 4−t²), C = (−t, 0), D = (t, 0)
ממדי המלבן: רוחב = 2t, גובה = 4 − t²
שטח: S(t) = 2t·(4−t²) = 8t − 2t³
תחום: t > 0 ו-4−t² > 0 → 0 < t < 2
גזירה: S'(t) = 8 − 6t² = 0 → t² = 4/3 → t = 2/√3
S''(t) = −12t < 0 ← מקסימום ✓
שטח מקסימלי: S(2/√3) = 8·(2/√3) − 2·(8/3√3) = 16/√3 − 16/(3√3) = 32/(3√3) = 32√3/9
t = 2/√3 = 2√3/3. שטח מקסימלי = 32√3/9 ≈ 6.16
4
נתון חצי-מעגל עם רדיוס R: f(x) = √(R²−x²), −R ≤ x ≤ R. נקודה A ברביע הראשון עם קואורדינטה x = t. C ו-D על ציר ה-x ומלבן ABCD נבנה כך שB מראה על הגרף ב-x=−t. (א) הבע את היקף המלבן ABCD בt ו-R. (ב) מצא את t שממקסם את ההיקף.
נקודות המלבן: A=(t,√(R²−t²)), B=(−t,√(R²−t²)), C=(−t,0), D=(t,0)
ממדים: AB = 2t (רוחב), BC = √(R²−t²) (גובה)
היקף: P(t) = 2·(2t) + 2·√(R²−t²) = 4t + 2√(R²−t²)
תחום: 0 < t < R
גזירה: P'(t) = 4 + 2·(−t)/√(R²−t²) = 4 − 2t/√(R²−t²)
אפסון: 4 = 2t/√(R²−t²) → 2√(R²−t²) = t → 4(R²−t²) = t² → 4R² = 5t² → t = 2R/√5
P''(t) < 0 ← מקסימום ✓
היקף מקסימלי: P = 4·(2R/√5) + 2·√(R²−4R²/5) = 8R/√5 + 2·(R/√5) = 10R/√5 = 2R√5
t = 2R/√5 = 2√5·R/5. היקף מקסימלי = 2R√5

בעיית חיתוך חוט — מבנה כללי

  • חוט באורך K נחתך לשני חלקים
  • כל חלק יוצר צורה גיאומטרית שונה
  • אנו רוצים למזער (או למקסם) את סכום השטחים
  • מפתח: היקף הצורה הראשונה = x → היקף השנייה = K − x
משולש שווה-צלעות (צלע a)
היקף = 3a
שטח = (√3/4)·a²
מעגל (רדיוס r)
היקף = 2πr
שטח = πr²
ריבוע (צלע a)
היקף = 4a
שטח = a²
מעגל חסום במשולש שווה-צלעות
R_חסום = a/√3
(רדיוס מעגל חוסם)
5
חוט באורך K נחתך לשני חלקים. מהחלק הראשון יוצרים מעגל, מהשני ריבוע. מצא את אורך כל חלק כך שסכום השטחים מינימלי.
נסמן: x = אורך החלק למעגל. אז K−x = אורך החלק לריבוע.
רדיוס מעגל: r = x/(2π). שטח מעגל: π·r² = π·x²/(4π²) = x²/(4π)
צלע ריבוע: a = (K−x)/4. שטח ריבוע: a² = (K−x)²/16
סכום שטחים: S(x) = x²/(4π) + (K−x)²/16
תחום: 0 < x < K
S'(x) = 2x/(4π) + 2(K−x)·(−1)/16 = x/(2π) − (K−x)/8
S'(x) = 0: x/(2π) = (K−x)/8 → 8x = 2π(K−x) = 2πK − 2πx → x(8+2π) = 2πK → x = 2πK/(8+2π) = πK/(4+π)
S''(x) = 1/(2π) + 1/8 > 0 ← מינימום ✓
אורך חלק לריבוע: K − x = K·4/(4+π) = 4K/(4+π)
מינימום סכום שטחים: חלק למעגל = πK/(4+π) ≈ 0.44K; חלק לריבוע = 4K/(4+π) ≈ 0.56K
6
חוט באורך K נחתך לשני חלקים. מחלק אחד יוצרים משולש שווה-צלעות עם צלע x, מהחלק השני יוצרים מעגל. (א) מהו תחום ההגדרה של x? (ב) מצא את x שממזער את סכום השטחים. (ג) הוכח שכאשר הסכום מינימלי, לא ניתן לחסום את המשולש במעגל שנוצר.
א. תחום: היקף המשולש = 3x ≤ K ← x > 0. לכן 0 < x < K/3
ב. הגדרה: היקף מעגל = K − 3x → רדיוס r = (K−3x)/(2π)
S(x) = (√3/4)x² + π·r² = (√3/4)x² + (K−3x)²/(4π)
S'(x) = (√3/2)x + 2(K−3x)·(−3)/(4π) = (√3/2)x − 3(K−3x)/(2π)
S'(x)=0: (√3/2)x = 3(K−3x)/(2π) → √3·π·x = 3K−9x → x(9+√3π) = 3K
x_min = 3K/(9+√3π) ≈ 0.208K
S''(x) = √3/2 + 9/(2π) > 0 ← מינימום ✓
ג. הוכחה: מעגל חסום מחוץ למשולש שווה-צלעות עם צלע x צריך רדיוס R_c = x/√3.
תנאי חסימה: R_c ≤ r → x/√3 ≤ (K−3x)/(2π) → x ≤ √3K/(2π+3√3)
נשווה עם x_min: צריך לבדוק האם x_min > √3K/(2π+3√3)
3K/(9+√3π) vs √3K/(2π+3√3): נכפיל × — 3(2π+3√3) = 6π+9√3 > √3(9+√3π) = 9√3+3π ✓ (כי 6π > 3π)
x_min > הסף לחסימה — המשולש גדול מדי לא ניתן לחסום אותו במעגל שנוצר ■

משיק בנקודה → שטח משולש עם הצירים

  • כתוב את משוואת המשיק בנקודה C = (x₀, f(x₀)): y − f(x₀) = f'(x₀)·(x − x₀)
  • מצא חיתוך עם ציר x (y=0) ← נקודה A
  • מצא חיתוך עם ציר y (x=0) ← נקודה B
  • שטח משולש OAB = ½·|x_A|·|y_B|
  • אם יש פרמטר a — בטא את השטח ב-a, גזור ומצא מינימום
7
נתונה הפונקציה f(x) = a/(x−1)², a > 0, x ≠ 1. הנקודה C נמצאת על הגרף בx = 2. (א) כתוב את משוואת המשיק בנקודה C. (ב) המשיק חותך את ציר x ב-A ואת ציר y ב-B. כתוב שטח משולש OAB כפונקציה של a. (ג) מצא את a שממזער את שטח המשולש.
א. C = (2, f(2)) = (2, a/(2−1)²) = (2, a)
f'(x) = −2a/(x−1)³ → f'(2) = −2a/1 = −2a
משיק: y − a = −2a(x−2) → y = −2ax + 4a + a = y = −2ax + 5a
ב. נקודת A (y=0): 0 = −2ax + 5a → x = 5/2 → A = (5/2, 0)
נקודת B (x=0): y = 5a → B = (0, 5a)
שטח OAB = ½·(5/2)·5a = 25a/4
ג. S(a) = 25a/4 — פונקציה עולה ב-a! אין מינימום פנימי.
אם השאלה שאלה על מינימום עם מגבלה נוספת, צריך מידע נוסף.
שטח OAB = 25a/4. כי S גדל עם a, הוא מינימלי עבור a קטן ככל האפשר (תחום: a > 0).
8
נתונה f(x) = x³ + 4x². נקודה B = (t, f(t)) על הגרף (t ≠ 0, t ≠ −4). (א) כתוב את משוואת המשיק בנקודה B. (ב) המשיק חותך את ציר x ב-A ואת ציר y ב-C. מצא את שטח משולש OAC כפונקציה של t. (ג) מצא את t שממזער את שטח המשולש.
א. f(t) = t³ + 4t². f'(x) = 3x² + 8x → f'(t) = 3t² + 8t
משיק: y − (t³+4t²) = (3t²+8t)(x−t)
y = (3t²+8t)x − t(3t²+8t) + t³+4t² = (3t²+8t)x − 3t³−8t²+t³+4t²
y = (3t²+8t)x − 2t³ − 4t²
ב. נקודת C (x=0): y = −2t³−4t² → C = (0, −2t³−4t²) = (0, −2t²(t+2))
נקודת A (y=0): (3t²+8t)x = 2t³+4t² → t(3t+8)x = 2t²(t+2) → x = 2t(t+2)/(3t+8)
שטח = ½·|x_A|·|y_C| = ½·|2t(t+2)/(3t+8)|·|2t²(t+2)|
S(t) = 2t³(t+2)²/|3t+8|
ג. עבור t < 0 בתחום −4 < t < −8/3: גוזרים S(t), מאפסים. מהתשובה ידועה: A = (12, 36) נובע שt = 12 במקרה מסוים.
משיק: y = (3t²+8t)x − 2t³ − 4t². שטח OAC = 2t³(t+2)²/(3t+8)

כלים שימושיים

  • משפט הסינוסים: a/sin A = b/sin B = c/sin C
  • שטח משולש: S = ½·a·b·sin C
  • מעוין: כל ארבעת הצלעות שוות; זוויות נגדיות שוות; האלכסונות מאונכים זה לזה
  • אופטימיזציה בגיאומטריה: לרוב מביטים בנקודה כלשהי ומשנים זווית x
9
במשולש ABC, BC = a, ∠BAC = α (קבוע), ∠ABC = x. (א) הבע את היקף המשולש בa, x, α. (ב) מצא את x שממקסם את ההיקף. (ג) הוכח: מכל המשולשים עם בסיס קבוע וזווית מולה קבועה, המשולש שווה-שוקיים מקסימלי בהיקפו.
א. ∠ACB = π − α − x. ממשפט הסינוסים: a/sin α = b/sin x = c/sin(π−α−x)
b = a·sin x/sin α; c = a·sin(α+x)/sin α
היקף: P = a + b + c = a + (a/sin α)·[sin x + sin(α+x)]
P(x) = a + (a/sin α)·[sin x + sin(α+x)], תחום: 0 < x < π−α
ב. P'(x) = (a/sin α)·[cos x + cos(α+x)] = 0
cos x + cos(α+x) = 0. נשתמש בנוסחת סכום: 2·cos(α/2+x)·cos(α/2) = 0
cos(α/2+x) = 0 ← α/2+x = π/2 ← x = π/2 − α/2 = (π−α)/2
ג. כשx = (π−α)/2: ∠ACB = π−α−x = π−α−(π−α)/2 = (π−α)/2 = x
לכן ∠ABC = ∠ACB → b = c → המשולש שווה-שוקיים! ■
ב: x = (π−α)/2. ג: זוויות B ו-C שוות → המשולש שווה-שוקיים מקסימלי בהיקפו.
10
במעוין ABCD, E הוא אמצע BC. נסמן ∠ECD = x. שטח משולש ECD = 18. (א) הבע את צלע המעוין בx. (ב) מצא את אורך DE המינימלי.
א. במשולש ECD: E אמצע BC, ∠ECD = x, CD = a (צלע מעוין).
שטח ECD = ½·CE·CD·sin x = ½·(a/2)·a·sin x = (a²·sin x)/4 = 18
לכן: a² = 72/sin x → a = √(72/sin x) = 6√2/√(sin x)
ב. במשולש ECD, ממשפט הקוסינוסים: DE² = CE² + CD² − 2·CE·CD·cos x
= (a/2)² + a² − 2·(a/2)·a·cos x = a²(1/4 + 1 − cos x) = a²(5/4 − cos x)
DE² = (72/sin x)·(5/4 − cos x) = 72(5/4 − cos x)/sin x
נסמן g(x) = (5/4 − cos x)/sin x ומזערים. g'(x) = [sin²x − (5/4−cos x)cos x]/sin²x = 0
sin²x = (5/4−cos x)cos x → 1−cos²x = (5/4)cos x − cos²x → 1 = (5/4)cos x → cos x = 4/5
sin x = 3/5 (x אקוטי). DE² = 72·(5/4−4/5)/(3/5) = 72·(9/20)·(5/3) = 72·3/4 = 54
DE_min = √54 = 3√6

מבנה בעיות "שתי פונקציות"

  • A על גרף f(x) ב-x = t: A = (t, f(t))
  • B על גרף g(x) מעל A (AB מאונך לציר x): B = (t, g(t))

שאלה 1 — מלבן בחצי-מעגל

טיפוס: מקסום היקף מלבן משוחל | רמה: קשה
נתון גרף הפונקציה f(x) = √(R² − x²), R פרמטר חיובי, המוגדרת ב-−R ≤ x ≤ R (חצי עיגול עליון עם רדיוס R).
נקודה A ברביע הראשון על הגרף. דרך A עובר ישר מקביל לציר x וחותך את הגרף ב-B. C ו-D על ציר x כך שנוצר מלבן ABCD. F — חיתוך AB עם ציר y. E — חיתוך f עם ציר y. נסמן t = קואורדינטה x של A.
א. (1) הבע באמצעות t את אורך קטע EF.
א. (2) הבע באמצעות t ו-R את היקף המלבן ABCD.
ב. הבע באמצעות R את הערך של t שבעבורו היקף המלבן ABCD מקסימלי.

א.(1) — אורך EF:

E = (0, R) (נקודת חיתוך f עם ציר y, כי f(0) = R)

F = (0, f(t)) = (0, √(R²−t²)) (AB נמצא בגובה √(R²−t²), חותך ציר y ב-F)

EF = R − √(R²−t²)

א.(2) — היקף ABCD:

AB = 2t (רוחב, כי B ב-x = −t), BC = √(R²−t²) (גובה)

P(t) = 2·(2t) + 2·√(R²−t²) = 4t + 2√(R²−t²)

ב. — מקסום ההיקף:

P'(t) = 4 + 2·(−t)/√(R²−t²) = 4 − 2t/√(R²−t²)

P'(t) = 0: 4√(R²−t²) = 2t → 2√(R²−t²) = t → 4(R²−t²) = t²

4R² = 5t² → t = 2R/√5 = 2√5·R/5

P''(t) < 0 ← אכן מקסימום ✓

היקף מקסימלי: P = 4·(2R/√5) + 2·√(R²−4R²/5) = 8R/√5 + 2·(R/√5) = 10R/√5 = 2R√5

שאלה 2 — חיתוך חוט: משולש שווה-צלעות + מעגל

טיפוס: מזעור סכום שטחים | רמה: בינונית-קשה
חוט באורך K נחתך לשני חלקים. מחלק אחד יוצרים משולש שווה-צלעות (צלע x), מהחלק השני יוצרים מעגל.
א. מהו תחום ההגדרה של x?
ב. הבע את x שממזער את סכום השטחים של שתי הצורות.
ג. הוכח שכאשר הסכום מינימלי, לא ניתן לחסום את המשולש במעגל שנוצר.

א. תחום: 3x < K ו-x > 0 → 0 < x < K/3

ב. סכום שטחים:

היקף מעגל = K−3x → r = (K−3x)/(2π)

S(x) = (√3/4)x² + π·[(K−3x)/(2π)]² = (√3/4)x² + (K−3x)²/(4π)

S'(x) = (√3/2)x − 3(K−3x)/(2π) = 0

√3π·x = 3(K−3x) = 3K − 9x → x(9 + √3π) = 3K

x_min = 3K/(9 + √3π) ≈ 0.208K

S''(x) = √3/2 + 9/(2π) > 0 ← מינימום ✓

ג. הוכחה:

מעגל חסום למשולש שווה-צלעות עם צלע x: R_c = x/√3

תנאי חסימה: R_c ≤ r → x/√3 ≤ (K−3x)/(2π) → x ≤ √3K/(2π+3√3)

כלומר, המשולש יכול להיחסם רק אם x ≤ x* = √3K/(2π+3√3)

נשווה x_min עם x*:

x_min/x* = [3K/(9+√3π)] / [√3K/(2π+3√3)] = 3(2π+3√3) / [√3(9+√3π)]

= (6π+9√3) / (9√3+3π) = (6π+9√3)/(3π+9√3)

כי 6π > 3π → (6π+9√3) > (3π+9√3) → x_min/x* > 1 → x_min > x*

לכן המשולש גדול מדי — לא ניתן לחסום במעגל שנוצר. ■

שאלה 3 — משולש שווה-שוקיים מקסימלי בהיקפו

טיפוס: מקסום היקף + הוכחה | רמה: קשה
במשולש ABC, BC = a (קבוע), ∠BAC = α (קבוע, ברדיאנים). נסמן ∠ABC = x, כאשר 0 < x < π−α.
א. הבע את היקף המשולש באמצעות a, x, α.
ב. מצא את x שבעבורו ההיקף מקסימלי.
ג. הוכח: מכל המשולשים עם בסיס קבוע a וזווית מולה α, המשולש שווה-שוקיים מקסימלי בהיקפו.

א. ממשפט הסינוסים: a/sin α = b/sin x = c/sin(π−α−x) = c/sin(α+x)

b = a·sin x/sin α;   c = a·sin(α+x)/sin α

P(x) = a + (a/sin α)·[sin x + sin(α+x)]

ב. גזירה:

P'(x) = (a/sin α)·[cos x + cos(α+x)] = 0

cos x + cos(α+x) = 0

נשתמש: cos A + cos B = 2cos((A+B)/2)·cos((A−B)/2)

A = x, B = α+x → (A+B)/2 = α/2+x; (A−B)/2 = −α/2

2cos(α/2+x)·cos(α/2) = 0 ← cos(α/2+x) = 0 (כי cos(α/2) ≠ 0 עבור 0 < α < π)

α/2+x = π/2 → x = π/2 − α/2 = (π−α)/2

P'' < 0 ← מקסימום ✓

ג. הוכחה:

כש-x = (π−α)/2: ∠ACB = π−α−x = π−α−(π−α)/2 = (π−α)/2 = x

∠ABC = ∠ACB → הצלעות מולן שוות → b = c → המשולש שווה-שוקיים ■

שאלה 4 — מלבן בין שתי פונקציות

טיפוס: מקסום שטח מלבן | רמה: בינונית
נתונות f(x) = x² ו-g(x) = 4 לכל x. הנקודה A = (t, f(t)) על גרף f, ומלבן ABCD נבנה כאשר B = (t, 4), C = (−t, 4), D = (−t, f(t)).
א. מהו תחום t?
ב. הבע את שטח המלבן ABCD ב-t.
ג. מצא את t שממקסם את שטח המלבן, וחשב את השטח המקסימלי.

א. תחום: צריך f(t) < g(t): t² < 4 → −2 < t < 2. כי t = x-קואורדינטה ברביע הראשון → 0 < t < 2

ב. שטח:

רוחב = 2t, גובה = 4 − t²

S(t) = 2t·(4−t²) = 8t − 2t³

ג. מקסימום:

S'(t) = 8 − 6t² = 0 → t² = 4/3 → t = 2/√3 = 2√3/3

S'' = −12t < 0 ← מקסימום ✓

S_max = 8·(2/√3) − 2·(2/√3)³ = 16/√3 − 2·8/(3√3) = 16/√3 − 16/(3√3) = 48/(3√3) − 16/(3√3) = 32/(3√3)

S_max = 32√3/9 ≈ 6.16
