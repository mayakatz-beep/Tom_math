כללי חיבור (איחוד אירועים)

אירועים זרים (שלא יכולים להתקיים ביחד)
A ∩ B = ∅   →   P(A ∪ B) = P(A) + P(B)
כלל ההסתברות הכולל (אירועים לא זרים)
P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
אם הבעיה נותנת P(A), P(B), P(A∪B) — אפשר לחשב P(A∩B) = P(A)+P(B)−P(A∪B).

= 18/30 + 12/30 − 5/30 = 25/30 = 5/6
✓ P(לפחות ספורט אחד) = 5/6

הסתברות מותנית — P(A|B)
הסיכוי שאירוע A יתקיים, בהינתן שאירוע B כבר התקיים.
B נקרא "התנאי" — הוא הופך להיות המרחב המדגם החדש שלנו.
נוסחת ההסתברות המותנית
P(A|B) = P(A ∩ B) / P(B)
בתנאי ש-P(B) > 0
איך לזהות הסתברות מותנית בשאלה? חפש מילים כמו:
"בהינתן ש...", "ידוע ש...", "נתון כי...", "מה הסיכוי שX, אם ידוע שY"
5
בכיתה יש 20 בנים ו-15 בנות. מבין הבנים 8 כסוחרים, מבין הבנות 5 כסוחרות.
נבחר תלמיד אקראי ונגלה שהוא כסוח. מה הסיכוי שהוא בן?
|Ω| = 35 תלמידים
B = "כסוח" → P(B) = (8+5)/35 = 13/35
A ∩ B = "בן וכסוח" → P(A∩B) = 8/35
P(A|B) = (8/35) ÷ (13/35) = 8/13
✓ P(בן | כסוח) = 8/13 ≈ 0.615

A = "גדול מ-3" = {4, 5, 6}

▼

כלל המכפלה נובע ישירות מהנוסחה של הסתברות מותנית — פשוט נכפיל שני הצדדים ב-P(B):

כלל המכפלה הכללי
P(A ∩ B) = P(B) · P(A|B)
= P(A) · P(B|A)
מתי להשתמש בכלל המכפלה?
כשרוצים לחשב סיכוי של שני דברים שקורים ביחד, ואחד תלוי בשני. לדוגמה: שליפה ללא החזרה, בחירה מתוך קבוצה שמשתנה אחרי כל שלב.
7
בשקית יש 5 כדורים אדומים ו-3 כחולים. שולפים שני כדורים ברצף ללא החזרה. מה הסיכוי שהראשון אדום וגם השני אדום?
A = "ראשון אדום" → P(A) = 5/8
B = "שני אדום" | ראשון אדום → נשארו 4 אדומים מתוך 7 → P(B|A) = 4/7
P(A ∩ B) = P(A) · P(B|A) = (5/8) · (4/7) = 20/56 = 5/14
✓ P(שניהם אדומים) = 5/14 ≈ 0.357
8
בקבוצה יש 4 בנים ו-6 בנות. בוחרים שניים בזה אחר זה ללא החזרה. מה הסיכוי שהנבחרים הם בן ואחר כך בת?
P(בן ראשון) = 4/10 = 2/5
P(בת שנייה | בן ראשון) = 6/9 = 2/3
P(בן, בת) = (2/5) · (2/3) = 4/15
✓ P(בן ואז בת) = 4/15 ≈ 0.267

עץ הסתברות
נושא 5 מתוך 7
▼
מתי בונים עץ?
כשהניסוי מתבצע בכמה שלבים עוקבים, ובכל שלב יש כמה אפשרויות. העץ עוזר לנו לראות את כל המסלולים האפשריים.

חוקי העץ

  • על כל ענף כותבים את ההסתברות המותנית של אותה אפשרות
  • הסיכוי של מסלול = מכפלת ההסתברויות לאורכו
  • הסיכוי של אירוע (שיכול לקרות בדרכים שונות) = סכום הסיכויים של המסלולים הרלוונטיים
  • סכום כל המסלולים = 1

דוגמה מלאה עם עץ — בעיית מפעל

9
במפעל יש 2 מכונות. מכונה A מייצרת 60% מהמוצרים, מכונה B מייצרת 40%. שיעור הפגמים: A → 3%, B → 5%. נבחר מוצר אקראי. מה הסיכוי שהוא פגום?
P(פגום) = P(A∩פגום) + P(B∩פגום)
= 0.6×0.03 + 0.4×0.05
= 0.018 + 0.020 = 0.038
בדיקה: סכום כל מסלולים = 0.018+0.582+0.020+0.380 = 1 ✓
✓ P(פגום) = 0.038 = 3.8%
10
שקית A: 3 אדומים, 2 כחולים. שקית B: 1 אדום, 4 כחולים. בוחרים שקית באקראי ושולפים כדור אחד. מה הסיכוי ששלפנו כחול?
P(A) = P(B) = 1/2
P(כחול|A) = 2/5,   P(כחול|B) = 4/5
P(כחול) = P(A)·P(כחול|A) + P(B)·P(כחול|B)
= (1/2)·(2/5) + (1/2)·(4/5) = 1/5 + 2/5 = 3/5
✓ P(כחול) = 3/5 = 0.6

אירועים בלתי תלויים
נושא 6 מתוך 7
▼
הגדרה: אירועים בלתי תלויים
A ו-B בלתי תלויים אם קיום A לא משפיע על הסיכוי של B, ולהיפך.
תנאי עצמאות
P(A ∩ B) = P(A) · P(B)
שקול ל: P(A|B) = P(A)   ושקול ל: P(B|A) = P(B)

כיצד לבדוק עצמאות?

  1. חשב P(A), P(B), P(A∩B)
  2. בדוק האם P(A∩B) = P(A)·P(B)
  3. אם כן — עצמאיים. אם לא — תלויים.
תלותיות ≠ זרות!
זרות: A∩B=∅ (לא יכולים להתקיים ביחד)
תלויות: קיום אחד משפיע על הסיכוי של השני
שני אירועים זרים (שאינם בעלי סיכוי 0) הם תמיד תלויים!
11
ידוע: P(A) = 0.4, P(B) = 0.3, P(A∩B) = 0.12. האם A ו-B בלתי תלויים?
בדיקה: P(A)·P(B) = 0.4 × 0.3 = 0.12
P(A∩B) = 0.12
0.12 = 0.12 ✓ → A ו-B בלתי תלויים
✓ כן, עצמאיים!
12
זורקים מטבע ואחר כך קוביה. A = "יצא עץ", B = "יצא 6 בקוביה". האם עצמאיים? חשב P(A∩B).
כן — שני ניסויים נפרדים, לכן ברור שהם בלתי תלויים.
P(A) = 1/2,   P(B) = 1/6
P(A∩B) = P(A)·P(B) = (1/2)·(1/6) = 1/12
✓ P(עץ וגם 6) = 1/12
13
שולפים קלף מחפיסה. A = "אס", B = "לב (♥)". האם עצמאיים?
P(A) = 4/52 = 1/13
P(B) = 13/52 = 1/4

P(A)·P(B) = (1/13)·(1/4) = 1/52
1/52 = 1/52 ✓ → עצמאיים! (הסוג והערך של קלף לא תלויים זה בזה)
✓ עצמאיים — P(אס ♥) = 1/52

משפט בייס
נושא 7 מתוך 7 | נושא מתקדם
▼
שאלת בייס — "החזרה לאחור"
בעיית בייס שואלת: ידוע תוצאה, מה הסיכוי שמקורה ב...?
לדוגמה: "המוצר יצא פגום — מה הסיכוי שמכונה A הפיקה אותו?"
משפט בייס
P(Aᵢ|B) = P(Aᵢ) · P(B|Aᵢ) / P(B)
כאשר P(B) = Σ P(Aⱼ)·P(B|Aⱼ) — סכום על כל מחלקות Aⱼ

שלבים לפתרון בייס — "נוסחת 3 השלבים"

  1. בנה עץ הסתברות (או טבלה)
  2. חשב P(B) = סכום כל מסלולי B (הסתברות כוללת)
  3. השתמש בנוסחה: P(Aᵢ|B) = מסלול Aᵢ → B חלקי P(B)
14
המשך דוגמה 9: נמצא מוצר פגום. מה הסיכוי שיוצר על ידי מכונה A?
מדוגמה 9: P(פגום) = 0.038
P(A ∩ פגום) = 0.6 × 0.03 = 0.018
P(A | פגום) = P(A ∩ פגום) / P(פגום) = 0.018 / 0.038
= 18/38 = 9/19 ≈ 0.474
✓ P(מכונה A | פגום) = 9/19 ≈ 47.4%
💡 שים לב: למרות שA מייצרת יותר (60%), שיעור הפגם שלה נמוך יותר (3% לעומת 5%) — לכן הסיכוי שFגום שלה הוא פחות מ-50%.
15
בבדיקה רפואית: 2% מהאוכלוסייה חולים. בדיקה חיובית ל-95% מהחולים, ל-8% מהבריאים. אדם נבדק וקיבל תוצאה חיובית. מה הסיכוי שהוא אכן חולה?
P(חולה) = 0.02,   P(בריא) = 0.98
P(חיובי|חולה) = 0.95,   P(חיובי|בריא) = 0.08
P(חיובי) = 0.02×0.95 + 0.98×0.08 = 0.019 + 0.0784 = 0.0974
P(חולה|חיובי) = (0.02×0.95) / 0.0974 = 0.019/0.0974 ≈ 0.195
✓ P(חולה | חיובי) ≈ 19.5%
💡 הפרדוקס הרפואי: למרות בדיקה מדויקת מאוד (95%), סיכוי שהאדם חולה הוא רק ~20% — כי המחלה נדירה מאוד.

P(A)·P(B) = 0.7×0.4 = 0.28 = P(A∩B) → עצמאיים!
2. בשקית 4 כדורים: 1 אדום, 3 כחולים. שולפים ללא החזרה שני כדורים. מה P(שניהם כחולים)?

3. ידוע P(B|A) = 0.6, P(A) = 0.5. מהו P(A∩B)?
P(A∩B) = P(A)·P(B|A) = 0.5×0.6 = 0.3
4. P(A) = 0.5, P(B) = 0.3. אם A ו-B עצמאיים, מהו P(Ā∩B)?
Ā ו-B גם עצמאיים! P(Ā)=0.5, P(Ā∩B)=0.5×0.3=0.15
5. ב-100 תלמידים: 60 בנים (15 מתוכם עם משקפיים), 40 בנות (10 עם משקפיים). נבחר תלמיד עם משקפיים. מה P(בת)?
P(בת∩משקפיים)=10/100=0.1, P(משקפיים)=25/100=0.25. P(בת|משקפיים)=0.1/0.25=10/25=2/5
6. מה הנוסחה הנכונה למשפט בייס?
P(A|B) = P(A)·P(B|A) / P(B) — זו נוסחת בייס הבסיסית.
7. P(A) = 0.4, P(Ā∩B) = 0.3, P(A∩B) = 0.1. מהו P(B)?
P(B) = P(A∩B) + P(Ā∩B) = 0.1 + 0.3 = 0.4

איך לגשת לשאלת בגרות מלאה?
1. קרא את השאלה כולה לפני שמתחילים לפתור
2. זהה מה מוגדר (מרחב, אירועים, ערכים נתונים)
3. ציין את כל הנתונים כסימונים ברורים
4. ענה על כל סעיף בסדר — כל אחד בונה על הקודם
שאלת בגרות 1
סקר פארק ביישוב | נושא: הסתברות מותנית + בייס

בביישוב הוחלט לערוך סקר לגבי הקמת פארק. בסקר השתתפו מבוגרים וצעירים. כל אחד מהתושבים שהשתתף כתב אם הוא תומך בהקמת הפארק או מתנגד (לא היו נמנעים). כל המבוגרים שהשתתפו בסקר תמכו בהקמת הפארק.

בחרו באקראי אחד מהתושבים שהשתתפו בסקר.

נסמן: p = ההסתברות שנבחר הוא צעיר  |  k = ההסתברות שנבחר תומך בפארק
✦ מחצית מהצעירים שהשתתפו בסקר תמכו בהקמת הפארק.
✦ 3/7 מהתושבים שתמכו בפארק היו צעירים.
א. הביעו באמצעות p ו-k את ההסתברות שנבחר היה צעיר תומך.
רעיון: כל המבוגרים תמכו → P(תמיכה ∩ מבוגר) = P(מבוגר) = 1−p
לכן: P(תמיכה) = P(צעיר תומך) + P(מבוגר תומך) = P(צעיר תומך) + (1−p)
k = P(צעיר תומך) + 1 − p
P(צעיר תומך) = k + p − 1
✓ P(צעיר ∩ תומך) = k + p − 1
ב. מצאו את p ואת k.
מהנתון: מחצית הצעירים תמכו → P(תומך | צעיר) = 1/2
→ P(צעיר תומך) = P(תומך|צעיר)·P(צעיר) = (1/2)·p = p/2
מהנתון: 3/7 מהתומכים הם צעירים → P(צעיר | תומך) = 3/7
→ P(צעיר תומך)/P(תומך) = 3/7 → (p/2)/k = 3/7 → k = 7p/6
שילוב עם תוצאת סעיף א: P(צעיר תומך) = k + p − 1 = p/2
7p/6 + p − 1 = p/2
7p/6 + 6p/6 − 3p/6 = 1 → 10p/6 = 1 → p = 3/5
k = 7p/6 = 7·(3/5)/6 = 21/30 = 7/10
בדיקה: P(צעיר תומך) = p/2 = 3/10. גם: k+p−1 = 7/10+3/5−1 = 7/10+6/10−10/10 = 3/10 ✓
✓ p = 3/5 (60% מהתושבים צעירים), k = 7/10 (70% תומכים בפארק)
ג. יוסי, כתב חדשות מקומי, ריאיין בזה אחר זה 5 תושבים שהשתתפו בסקר.
מה ההסתברות שבדיוק 3 מהם היו צעירים, כאשר המרואיין האחרון היה צעיר?
התנאי: המרואיין ה-5 הוא צעיר. אז בין 4 הראשונים יש בדיוק 2 צעירים.
מבנה: בחר אילו 2 מתוך 4 הראשונים הם צעירים → C(4,2) = 6 דרכים
p = P(צעיר) = 3/5, P(מבוגר) = 2/5
P(מסלול ספציפי: ז,ז,מ,מ,ז) = (3/5)²·(2/5)²·(3/5) = 9/25 · 4/25 · 3/5 = 108/3125
P(כל מסלולי "בדיוק 2 ראשונים צעירים + ה-5 צעיר") = C(4,2) · 108/3125 = 6 · 108/3125 = 648/3125
✓ P = 648/3125
שאלת בגרות 2
ספורט בבית ספר | נושא: עצמאות + הסתברות מותנית

ערכו סקר בין תלמידי בית ספר. 80% מהבנים עוסקים בספורט.
מספר הבנות בבית הספר גדול פי 1.2 ממספר הבנים שעוסקים בספורט.

א. מהי ההסתברות שתלמיד שנבחר אקראית הוא בן?
נסמן: B = מספר הבנים, G = מספר הבנות
בנים שעוסקים בספורט = 0.8B
מהנתון: G = 1.2 × 0.8B = 0.96B
סה"כ תלמידים = B + 0.96B = 1.96B
P(בן) = B / 1.96B = 1/1.96 = 100/196 = 25/49
✓ P(בן) = 25/49 ≈ 51%
ב. מהי ההסתברות שנבחר תלמיד העוסק בספורט, אם ידוע שהמאורעות "נבחר בן" ו"נבחר ספורטאי" הם בלתי תלויים? הסבר.
נסמן: s = P(ספורטאי). עצמאות ↔ P(בן ∩ ספורטאי) = P(בן) · P(ספורטאי)
P(בן ∩ ספורטאי) = P(ספורטאי|בן)·P(בן) = 0.8 · (25/49) = 20/49
מעצמאות: 20/49 = (25/49) · s → s = (20/49) ÷ (25/49) = 20/25 = 4/5
כלומר: 80% מהתלמידים עוסקים בספורט.
הסבר: P(ספורטאי|בן) = 0.8 = P(ספורטאי|בת)? בדיקה: P(בת) = 24/49. P(בת∩ספורטאי) = 4/5 − 20/49 = 96/245. P(ספורטאי|בת) = (96/245)/(24/49) = 4/5 ✓ שווה! אכן בלתי תלויים.
✓ P(ספורטאי) = 4/5 = 80%
ג. נבחר באקראי תלמיד העוסק בספורט. מה סביר יותר — שהוא בן או בת? הסבר.
P(בן | ספורטאי) = P(בן ∩ ספורטאי) / P(ספורטאי) = (20/49) / (4/5) = (20/49) · (5/4) = 100/196 = 25/49
P(בת | ספורטאי) = P(בת ∩ ספורטאי) / P(ספורטאי) = (96/245) / (4/5) = (96/245)·(5/4) = 480/980 = 24/49
25/49 > 24/49 → סביר יותר שהוא בן.
💡 הסבר אינטואיטיבי: יש יותר בנים (25/49) מבנות (24/49) בבית הספר, ושניהם עוסקים בספורט באותו שיעור (80%) — לכן גם בין הספורטאים יש יותר בנים.
✓ סביר יותר שהוא בן: P(בן|ספורטאי) = 25/49 > P(בת|ספורטאי) = 24/49
שאלת בגרות 3
תחרות ספורט | נושא: הסתברות מותנית + עץ + כלל המכפלה

בתחרות ספורט שנערכה בבית ספר השתתפו תלמידים רבים. כל משתתף חייב לעבור 3 מכשולים בזה אחר זה לפי הסדר. משתתף שנכשל במכשול מודח מייד. ההסתברות לעבור כל מכשול שונה ממכשול למכשול, אך שווה לכל המשתתפים. משתתף שעובר את המכשול הראשון עולה לחצי-גמר.

✦ 28% מהמשתתפים עברו את שני המכשולים הראשונים.
✦ הסיכוי שמי שעבר שני מכשולים יודח (לא יעבור מכשול 3) הוא פי 3 מהסיכוי שיעלה לגמר.
✦ ההסתברות לעבור מכשול 1 ולהיכשל במכשול 2 היא 0.42.
א. חשב את ההסתברות שמשתתף עובר מכשול 1 (עולה לחצי-גמר).
נסמן: p₁, p₂, p₃ = הסתברות לעבור מכשול 1, 2, 3 בהתאמה
נתון א: P(עבר 1 וגם 2) = p₁·p₂ = 0.28
נתון ב: P(לא עבר 3 | עבר 1+2) = 3·P(עבר 3 | עבר 1+2)
← (1−p₃) = 3p₃ → p₃ = 1/4
נתון ג: P(עבר 1, נכשל 2) = p₁·(1−p₂) = 0.42
מנתונים א+ג: p₁·p₂ + p₁·(1−p₂) = 0.28 + 0.42 = 0.70 → p₁ = 0.70
✓ P(חצי-גמר) = p₁ = 0.70
ב. חשב את ההסתברות שמשתתף לא עובר מכשול 1.
P(לא עבר מכשול 1) = 1 − p₁ = 1 − 0.70 = 0.30
✓ P(נכשל במכשול 1) = 0.30
ג. נבחרו באקראי שלושה משתתפים: עומר, גל וליאור. כולם עברו מכשול 1.
p₂ = 0.28/0.70 = 0.40
(1) חשב את ההסתברות שבדיוק 2 מהשלושה יגיעו לגמר (יעברו גם מכשול 2).
(2) חשב את ההסתברות שדווקא עומר וגל יגיעו לגמר, אבל ליאור לא.
p₂ = 0.28 / 0.70 = 0.40 (הסתברות לעבור מכשול 2)
(1) בדיוק 2 מ-3 עוברים מכשול 2 (ניסויים בלתי תלויים):
P = C(3,2) · (0.4)² · (0.6)¹ = 3 · 0.16 · 0.6 = 0.288
(2) דווקא עומר וגל עוברים (וליאור לא):
P = P(עומר עובר) · P(גל עובר) · P(ליאור לא עובר)
= 0.4 · 0.4 · 0.6 = 0.096
✓ (1) P(בדיוק 2 עוברים) = 0.288    (2) P(רק עומר+גל) = 0.096
