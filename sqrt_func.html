√ "הגבול הבלתי נראה" — מדוע השורש דורש תנאי

שורש ריבועי של מספר שלילי לא קיים במספרים ממשיים. לכן כל פונקציה עם שורש מביאה עמה "תנאי קיום" — הדרישה שתוכן השורש יהיה אי-שלילי. זה מגביל את תחום ההגדרה.

שורש במונה: √(ביטוי) מוגדר כש-ביטוי ≥ 0. כולל אפס — כי √0 = 0 זה בסדר.
שורש במכנה: √(ביטוי) במכנה → ביטוי > 0 (חמור!). √0 במכנה = חלוקה ב-0 = אסור.
שיטה לפתרון: רשום אי-שוויון → פתור → ציר מספרים עם נקודות חלוקה → בדוק סימן בכל תחום.

כשיש כמה תנאים ביחד (שורש ומכנה, שני שורשים) — תחום ההגדרה הוא חיתוך של כל התנאים. כל תנאי חייב להתקיים בו-זמנית.

שים לב לסימטריה: אם f(x) = √(a² − x²) — זה חצי-מעגל! תחום: −a ≤ x ≤ a.

כלל בסיסי – תנאי קיום שורש ריבועי

  • √(ביטוי) מוגדר כאשר הביטוי ≥ 0
  • אם השורש במכנה: הביטוי חייב להיות > 0 (חוסר שוויון חמור)
  • אם יש גם שורש וגם מכנה — משלבים את כל התנאים
  • שיטה: ציר מספרים, בדיקת סימנים לאי-שוויון

√x : תחום x ≥ 0

נוסחאות נגזרת – שורש ריבועי

  • (√x)' = 1/(2√x)
  • (√(f(x)))' = f'(x) / (2√(f(x))) — כלל השרשרת
  • לנגזרת מנה: [p/q]' = (p'q − pq') / q²
  • חלופה נוחה: לכתוב 1/√x = x^(−1/2) ולגזור כחזקה

מצאו את f'(x) עבור f(x) = √x/(x+1)
כלל מנה: p = √x, q = x+1
p' = 1/(2√x),   q' = 1
f'(x) = [(1/(2√x))·(x+1) − √x·1] / (x+1)²
פישוט מונה — מכנה משותף 2√x:
מונה = (x+1)/(2√x) − √x = [(x+1) − 2x] / (2√x) = (1−x)/(2√x)

שגיאה נפוצה: לשכוח לפשט את המונה לאחר כלל המנה. תמיד כדאי לצמצם ל-מכנה משותף אחד.

שאלות סוג "השוואה" בבגרות

  • נתונה f(x) ועוד g(x), h(x) המוגדרות דרכה: g(x) = [f(x)]², h(x) = 1/f(x)
  • זיהוי גרף מתוך כמה אפשרויות — בדקו: תחום, קיצון, אסימפטוטות, נקודות מסוימות
  • הזזות: g(x) = f(x−2) → הזזה ימינה ב-2; h(x) = f(x)+3 → הזזה למעלה ב-3
  • f''(x): אם f''(x) > 0 → קמורה (∪); אם f''(x) < 0 → קעורה (∩)
  • פונקציה זוגית: f(−x) = f(x) — סימטרית לציר y
