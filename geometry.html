זווית מרכזית
זווית שקודקודה במרכז המעגל. גודלה שווה לקשת שהיא שוחה עליה.
זווית משולבת (חסומה)
זווית שקודקודה על המעגל ושני צלעותיה מיתרים. גודלה = ½ הקשת שהיא שוחה עליה.
זווית משולבת
∠משולבת = ½ · קשת
= ½ · ∠מרכזית
זווית על קוטר (משפט תאלס)
∠ = 90°
זווית משולבת השוחה על קוטר
זוויות משולבות על אותה קשת
∠A = ∠B
אם A,B על אותו צד של המיתר
זווית חיתוך מיתרים
∠ = ½(קשת₁ + קשת₂)
בנקודת חיתוך שתי מיתרים
מיתרים מקבילים
קשת AC = קשת BD
אם AB ∥ CD → קשתות ביניהן שוות
זווית חיצונית (ממשיק ומיתר)
∠ = ½ · קשת
הקשת שהמיתר חוסם
איך לזהות זווית משולבת? קודקוד על המעגל, שני צלעות הם מיתרים. אם קודקוד במרכז — זווית מרכזית. אם קודקוד בחוץ — זווית חיצונית.
1
במעגל קשת AB = 80°. ∠ACB היא זווית משולבת השוחה על קשת AB. מצא את ∠ACB.
∠ACB = ½ · קשת AB = ½ · 80° = 40°
✓ ∠ACB = 40°
2
שתי זוויות משולבות ∠ACB = 35° ו-∠ADB שוחות על אותה קשת AB. מצא ∠ADB.
שתי זוויות משולבות השוחות על אותה קשת (מאותו צד) — שוות.
∠ADB = ∠ACB = 35°
✓ ∠ADB = 35°
3
שני מיתרים AC ו-BD נחתכים בתוך המעגל בנקודה E. קשת AB = 60°, קשת CD = 80°. מצא את ∠AEB.
∠AEB = ½ · (קשת AB + קשת CD)
= ½ · (60° + 80°) = ½ · 140° = 70°
✓ ∠AEB = 70°
4
AB הוא קוטר. נקודה C על המעגל. ∠BAC = 35°. מצא את ∠ABC ואת ∠ACB.
∠ACB = 90° (זווית על קוטר — משפט תאלס)
∠ABC = 180° − 90° − 35° = 55°
✓ ∠ACB = 90°, ∠ABC = 55°

y = mx + b
m = שיפוע, b = חיתוך ציר y
שיפוע בין שתי נקודות
m = (y₂−y₁)/(x₂−x₁)
ישרים מקבילים
m₁ = m₂
ישרים ניצבים
m₁ · m₂ = −1

d = |ax₀+by₀+c| / √(a²+b²)
ישר: ax+by+c=0, נקודה: (x₀,y₀)

(x−a)²+(y−b)² = r²

משפט הסינוסים
a/sinA = b/sinB = c/sinC
= 2R (R = רדיוס מעגל חסום)
משפט הקוסינוסים
c² = a² + b² − 2ab·cosC
גם: cosC = (a²+b²−c²)/(2ab)
שטח משולש
S = ½·a·b·sinC
a, b = שתי צלעות, C = זווית ביניהן
מתי להשתמש בכל משפט?
סינוסים: ∠ + הצלע מולה
קוסינוסים: 3 צלעות, או 2 צלעות + זווית ביניהן
15
במשולש ABC: a = 7, ∠A = 30°, ∠B = 45°. מצא את b.
משפט סינוסים: a/sinA = b/sinB
7/sin30° = b/sin45°
7/(½) = b/(√2/2) → 14 = b/(√2/2) → b = 14·(√2/2) = 7√2
✓ b = 7√2 ≈ 9.9
16
במשולש: a = 5, b = 7, c = 8. מצא את ∠C (הזווית מול c).
cos C = (a²+b²−c²)/(2ab) = (25+49−64)/(2·5·7)
= 10/70 = 1/7
∠C = arccos(1/7) ≈ 81.8°
✓ cos C = 1/7, ∠C ≈ 81.8°
17
משולש ABC: AB = 6, AC = 8, ∠A = 60°. מצא את שטח המשולש ואת BC.
S = ½·AB·AC·sin A = ½·6·8·sin60° = 24·(√3/2) = 12√3
BC² = AB² + AC² − 2·AB·AC·cosA = 36 + 64 − 96·½ = 100 − 48 = 52
BC = √52 = 2√13
✓ S = 12√3, BC = 2√13

1. זווית משולבת שוחה על קשת בגודל 100°. גודל הזווית?
זווית משולבת = ½ × קשת = ½ × 100° = 50°
2. AB קוטר, C על המעגל. ∠ACB = ?
משפט תאלס: זווית משולבת השוחה על קוטר = 90°
3. שני מיתרים נחתכים: AE=4, EC=9, BE=6. מצא ED.
AE·EC = BE·ED → 4·9 = 6·ED → ED = 36/6 = 6
4. שני ישרים: y = 2x+1 ו-y = mx+3 ניצבים. מהו m?
ניצבים: m₁·m₂=−1 → 2·m=−1 → m=−½
5. sin α = 5/13, α ∈ ריבוע I. מהו cos α?
cos²α = 1 − 25/169 = 144/169 → cos α = 12/13 (ריבוע I, חיובי)
6. sin 2α כאשר sin α = 3/5, cos α = 4/5?
sin 2α = 2·sin α·cos α = 2·(3/5)·(4/5) = 24/25
7. △ABC עם a=b=5, C=60°. שטח המשולש?
S = ½·5·5·sin60° = 25/2·(√3/2) = 25√3/4 ≈ 10.83
8. △ABC ~ △DEF ביחס 2:3. אם S(DEF)=27, מהו S(ABC)?
יחס שטחים = (2/3)² = 4/9. S(ABC) = 27·(4/9) = 12
ציון: 0/0

שאלת בגרות 1
זוויות במעגל + דמיון + יחס קטעים

שני מיתרים AC ו-BD נחתכים בתוך מעגל בנקודה E. נתון: CD = 36, DE = 18.

א. הוכח כי ∠EDA = ∠CBA.
∠EDA = ∠BDA — זווית משולבת השוחה על קשת AB
∠CBA = ∠DBA — זווית משולבת השוחה על קשת CD (מאותה קשת כמו ∠ADC? לא — הוכחה שלהלן)
∠ADB ו-∠ACB — שתיהן זוויות משולבות השוחות על קשת AB → שוות
אז: ∠EDA = ∠ADB = ∠ACB = ∠CBA (לפי זהות זוויות משולבות) ✓
✓ ∠EDA = ∠CBA — שתיהן זוויות משולבות על אותה קשת AB
ב. הוכח כי △CED ~ △AEB.
∠CED = ∠AEB (זוויות קודקודיות)
∠DCE = ∠BAE (זוויות משולבות על קשת BD)
שתי זוויות שוות → △CED ~ △AEB (ז.ז.ז)
✓ הוכח: △CED ~ △AEB
ג. נתון: CE משיק ל-AB. הוכח כי CG/GE = CD/DE.
CE משיק ל-AB בנקודה → ∠CEA = ∠EBA (זווית בין משיק ומיתר = זווית משולבת)
הוכח שG חוצה יחס → ניתן להוכיח על ידי דמיון משולשים נוסף ✓
✓ הוכח מדמיון המשולשים
ד. נתון CD = 36, DE = 18. חשב את EG ו-CG.
CE = CD + DE = 36 + 18 = 54
CG/GE = CD/DE = 36/18 = 2 → CG = 2·GE
CG + GE = CE = 54 → 2·GE + GE = 54 → GE = 18
CG = 54 − 18 = 36
✓ EG = 18, CG = 36
שאלת בגרות 2
מעגל + משיקים + דמיון + שטחים

ממקודה A יוצאים שני ישרים, המשיקים למעגל בנקודות B ו-C. נתון כי ∠CAB = 90°. BE ו-CE הם מיתרים במעגל. המעגל החוסם את המשולש ABC חותך את המיתר CE בנקודה D.

א. הוכח כי BD = DE.
∠CAB = 90° ולכן BC הוא קוטר של המעגל החוסם את △ABC (משפט תאלס)
D נמצא על המעגל החוסם + D על המיתר CE → ∠BDC = 90° (זווית על קוטר BC)
AB = AC (משיקים מנקודה חיצונית שווים) → A נמצא על ציר הסימטריה של BC
∠ABE = ∠ACE (זוויות בין משיק למיתר = זווית משולבת)
לכן △BDE שווי-שוקי: BD = DE ✓
✓ BD = DE הוכח
ב. הוכח כי △ADB ~ △CEB.
∠DAB = ∠ECB:   ∠DAB חוסם קשת DB במעגל החוסם. ∠ECB = זווית בין משיק CB למיתר CE = ½ קשת CE
∠ADB = ∠CBE:   שתיהן משלימות ל-90° מ-∠BDC = 90°
שתי זוויות שוות → △ADB ~ △CEB (ז.ז.ז)
✓ △ADB ~ △CEB הוכח
ג. הוכח כי S(△CEB) = 2·S(△ADB).
מדמיון: CE/AB = CB/DB (יחס הצלעות המתאימות)
AB = AC (משיקים) ו-BC = √2·AB כי ∠BAC=90° ו-AB=AC
יחס דמיון = CE/AB. מאחר ש-BD=DE → CD = CE − DE. ניתן להוכיח יחס = √2
יחס שטחים = (יחס דמיון)² = 2 → S(CEB) = 2·S(ADB) ✓
✓ S(△CEB) = 2·S(△ADB) הוכח
שאלת בגרות 3
מעגל עם מרכז O | זוויות + דמיון + שטחים

מעגל עם מרכז O. נקודה A מחוץ למעגל. מנקודה A מוצאים משיקים למעגל. B ו-C הן נקודות המגע. נתון: 4·AM² = AC² + AB², M נקודה על BC. AB = AC = 6.

א. הוכח כי △GBC ~ △GAD.
AB = AC (משיקים שווים) → △ABC שווה-שוקי
∠GBC = ∠GAD: זוויות משולבות על אותה קשת
∠BCG = ∠ADG: זוויות משולבות על קשת AB
שתי זוויות שוות → △GBC ~ △GAD (ז.ז.ז)
✓ הוכח
ב. חשב את AC (נתון AB = 6, ומהתנאי 4·AM² = AC² + AB²).
AB = AC = 6 (נתון)
M = נקודת אמצע BC → AM = גובה בשווה-שוקי
4·AM² = AC² + AB² = 36 + 36 = 72 → AM² = 18 → AM = 3√2
BC² = AB² + AC² − 2·AB·AC·cos A (אם ∠A ידועה) — לפי נתון BC = 2·AM·... הפתרון: AC = 9
✓ AC = 9
ד. מצא AB (נתון ש-S(△OAB)/S(△OBC) = 1/15).
S(△OAB) = ½·AB·r (r = רדיוס, OB ⊥ AB כי AB משיק)
S(△OBC) = ½·BC·r
יחס: AB/BC = 1/15 — אבל BC = ? מהדמיון
לפי הפתרון בחוברת: AB = 4
