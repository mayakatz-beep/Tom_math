Чҗ. ЧһЧҰЧҗЧ• ЧҗЧӘ ЧўЧЁЧҡ Ч”ЧӨЧЁЧһЧҳЧЁ a.
Ч‘. Ч”ЧҰЧҷЧ‘Ч• a=1 Ч•ЧўЧ‘Ч“Ч• Ч‘ЧӘЧ—Ч•Чқ вҲ’2ПҖ вүӨ x вүӨ 2ПҖ. ЧһЧҰЧҗЧ•: (1) ЧӘЧ—Ч•Чқ Ч”Ч’Ч“ЧЁЧ”; (2) Ч Ч§Ч•Ч“Ч•ЧӘ Ч—ЧҷЧӘЧ•Чҡ ЧўЧқ Ч”ЧҰЧҷЧЁЧҷЧқ; (3) Ч Ч§Ч•Ч“Ч•ЧӘ Ч§ЧҷЧҰЧ•Чҹ.
Ч’. ЧЎЧЁЧҳЧҳЧ• ЧЎЧ§ЧҷЧҰЧ”.
Ч“. ЧһЧҰЧҗЧ• ЧҗЧӘ ЧһЧЎЧӨЧЁ Ч”ЧӨЧӘЧЁЧ•Ч Ч•ЧӘ Ч©Чң f(x) = вҲ’1.

Ч‘Ч Ч§Ч•Ч“Ч•ЧӘ Ч§ЧҷЧҰЧ•Чҹ f(x) = 0 (ЧЁЧҷЧҰЧӨЧ” ЧңЧҰЧҷЧЁ x):

(3) Ч Ч§Ч•Ч“Ч•ЧӘ Ч§ЧҷЧҰЧ•Чҹ: f'=0 вҶ” cos x=0
ЧһЧ§ЧЎЧҷЧһЧ•ЧһЧҷЧқ Ч‘-(вҲ’3ПҖ/2, 0) Ч•-(ПҖ/2, 0)
Ч’. ЧЎЧ§ЧҷЧҰЧ”: ЧҗЧЎЧҷЧһЧӨЧҳЧ•ЧҳЧ•ЧӘ ЧҗЧ ЧӣЧҷЧ•ЧӘ Ч‘-x=вҲ’ПҖ/2 Ч•-x=3ПҖ/2. fвҶ’вҲ’вҲһ ЧңЧҷЧ“ Ч”ЧҗЧЎЧҷЧһЧӨЧҳЧ•ЧҳЧ•ЧӘ. Ч©Ч Чҷ ЧһЧ§ЧЎЧҷЧһЧ•ЧһЧҷЧқ Ч‘-y=0. f(0)=вҲ’1.

5 ЧӨЧӘЧЁЧ•Ч Ч•ЧӘ
Ч”. Ч”Ч•ЧӣЧ—Ч” Ч©-вҲ«вӮҖ^ПҖ (f(x)+1) dx > ПҖ/2:
f(x)+1 = (sin xвҲ’1)/(sin x+1) + 1 = (sin xвҲ’1+sin x+1)/(sin x+1) = 2sin x/(sin x+1)
ЧўЧң (0,ПҖ): sin x > 0 Ч•ЧңЧӣЧҹ f(x)+1 > 0 вң“
f(ПҖвҲ’x)+1 = 2sin(ПҖвҲ’x)/(sin(ПҖвҲ’x)+1) = 2sin x/(sin x+1) = f(x)+1 вҶ’ ЧЎЧҷЧһЧҳЧЁЧҷЧ” Ч‘-ПҖ/2
Ч‘Ч Ч§Ч•Ч“Ч” x=ПҖ/4: f(ПҖ/4)+1 = 2В·(вҲҡ2/2)/((вҲҡ2/2)+1) = вҲҡ2/(1+вҲҡ2/2) > 1/2

вҲ«вӮҖ^ПҖ (f(x)+1) dx > 0. ЧҗЧҡ Ч Ч“Ч’ЧҷЧқ Ч—ЧЎЧқ Ч ЧһЧ•Чҡ ЧҷЧ•ЧӘЧЁ:
Ч‘ЧӘЧ—Ч•Чқ [ПҖ/4, 3ПҖ/4]: sin x вүҘ вҲҡ2/2, ЧңЧӣЧҹ f(x)+1 = 2sin x/(sin x+1) вүҘ 2В·(вҲҡ2/2)/(вҲҡ2/2+1) = вҲҡ2/(1+вҲҡ2/2) вүҲ 0.828 > ПҖ/(2ПҖ) ...
Ч©ЧҷЧҳЧ” ЧҷЧ©ЧҷЧЁЧ”: вҲ«вӮҖ^(ПҖ/2)(f(x)+1)dx = вҲ«вӮҖ^(ПҖ/2)2sin x/(sin x+1)dx. Ч‘-[0,ПҖ/2]: sin x/(sin x+1) вүҘ sin x/2 (ЧӣЧҷ sin x+1 вүӨ 2), ЧңЧӣЧҹ 2sin x/(sin x+1) вүҘ sin x.

Ч ЧӘЧ•Ч Ч” Ч”ЧӨЧ•Ч Ч§ЧҰЧҷЧ” f(x) = cos x / вҲҡ(sin x) Ч‘ЧӘЧ—Ч•Чқ 0 < x < 2ПҖ.

Чҗ. ЧһЧҰЧҗЧ• ЧҗЧӘ ЧӘЧ—Ч•Чқ Ч”Ч”Ч’Ч“ЧЁЧ” Ч©Чң f(x).
Ч‘. ЧһЧҰЧҗЧ• ЧҗЧӘ Ч Ч§Ч•Ч“Ч•ЧӘ Ч”Ч§ЧҷЧҰЧ•Чҹ Ч©Чң f(x) (ЧҗЧқ ЧҷЧ©).
Ч’. ЧһЧҰЧҗЧ• ЧҗЧӘ Ч Ч§Ч•Ч“Ч•ЧӘ Ч”ЧҗЧӨЧЎ Ч©Чң f(x).
Ч“. ЧЎЧЁЧҳЧҳЧ• ЧЎЧ§ЧҷЧҰЧ” Ч©Чң Ч”Ч’ЧЁЧЈ.
Чҗ. ЧӘЧ—Ч•Чқ Ч”Ч’Ч“ЧЁЧ”:
Ч“ЧЁЧ•Ч©: вҲҡ(sin x) вү  0 (Ч‘ЧһЧӣЧ Ч”) Ч•Ч’Чқ sin x вүҘ 0 вҶ’ sin x > 0
sin x > 0 Ч‘ЧӘЧ—Ч•Чқ (0, 2ПҖ): x вҲҲ (0, ПҖ)
ЧӘЧ—Ч•Чқ Ч”Ч”Ч’Ч“ЧЁЧ”: (0, ПҖ)
Ч‘. Ч Ч’Ч–ЧЁЧӘ:
f(x) = cos x В· (sin x)^(вҲ’1/2)
f'(x) = вҲ’sin x В· (sin x)^(вҲ’1/2) + cos x В· (вҲ’1/2)(sin x)^(вҲ’3/2) В· cos x
= вҲ’sin x/вҲҡ(sin x) вҲ’ cosВІx/(2sin x В· вҲҡ(sin x))
= [вҲ’2sinВІx вҲ’ cosВІx] / (2sin x В· вҲҡ(sin x))
= вҲ’(sinВІx + sinВІx + cosВІx) / (2sin x В· вҲҡ(sin x))
= вҲ’(sinВІx + 1) / (2sin x В· вҲҡ(sin x))
ЧһЧӣЧҷЧ•Ч•Чҹ Ч©sinВІx + 1 > 0 Ч•sin x > 0 Ч‘ЧӘЧ—Ч•Чқ: f'(x) < 0 ЧӘЧһЧҷЧ“!
ЧҗЧҷЧҹ Ч Ч§Ч•Ч“Ч•ЧӘ Ч§ЧҷЧҰЧ•Чҹ вҖ” f ЧҷЧ•ЧЁЧ“ЧӘ ЧһЧ•Ч Ч•ЧҳЧ•Ч ЧҷЧӘ ЧўЧң ЧӣЧң (0, ПҖ)
Ч’. ЧҗЧӨЧЎЧҷ Ч”ЧӨЧ•Ч Ч§ЧҰЧҷЧ”:
f(x) = 0 вҶ” cos x = 0 (sin x > 0 Ч‘ЧӘЧ—Ч•Чқ)
cos x = 0 Ч‘-x = ПҖ/2
Ч Ч§Ч•Ч“ЧӘ ЧҗЧӨЧЎ: (ПҖ/2, 0)
Ч“. ЧһЧҗЧӨЧҷЧҷЧ Чҷ Ч”Ч’ЧЁЧЈ:
вҖў xвҶ’0вҒә: cos x вҶ’ 1, sin x вҶ’ 0вҒә вҶ’ f(x) вҶ’ +вҲһ (ЧҗЧЎЧҷЧһЧӨЧҳЧ•ЧҳЧ” ЧҗЧ ЧӣЧҷЧӘ)
