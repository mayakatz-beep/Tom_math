סדרה חשבונית (Arithmetic)
נושא 1 מתוך 7
▼
מה זו סדרה חשבונית?
סדרה שבה ההפרש בין כל שני איברים עוקבים קבוע ושווה. ההפרש הקבוע נסמן d.
איבר כללי — סדרה חשבונית
aₙ = a₁ + (n−1)·d
a₁ = האיבר הראשון  |  d = ההפרש  |  n = מספר האיבר
כדי למצוא d: קח כל שני איברים עוקבים → d = aₙ₊₁ − aₙ

תכונות מרכזיות

  • d > 0 → סדרה עולה
  • d < 0 → סדרה יורדת
  • d = 0 → סדרה קבועה
  • האיבר האמצעי של שלושה עוקבים = ממוצע שני הקצוות: aₙ = (aₙ₋₁ + aₙ₊₁)/2
1
בסדרה חשבונית: a₁ = 3, d = 5. מצא את a₁₀ ואת n עבור aₙ = 48.
aₙ = a₁ + (n−1)·d = 3 + (n−1)·5
a₁₀ = 3 + 9·5 = 3 + 45 = 48
aₙ = 48:   3 + (n−1)·5 = 48 → (n−1)·5 = 45 → n−1 = 9 → n = 10
✓ a₁₀ = 48, n = 10
2
בסדרה חשבונית: a₃ = 11, a₇ = 27. מצא את a₁ ואת d.
a₃ = a₁ + 2d = 11    (משוואה I)
a₇ = a₁ + 6d = 27    (משוואה II)
II − I:   4d = 16 → d = 4
מ-I: a₁ = 11 − 2·4 = 3
✓ a₁ = 3, d = 4
3
בסדרה חשבונית: a₁ = 100, d = −7. מהו האיבר השלילי הראשון?
aₙ = 100 + (n−1)·(−7) < 0
100 − 7(n−1) < 0 → 7(n−1) > 100 → n−1 > 100/7 ≈ 14.28
n > 15.28 → n = 16 (מינימום שלם)
a₁₆ = 100 + 15·(−7) = 100 − 105 = −5
✓ האיבר השלילי הראשון הוא a₁₆ = −5
➕
סכום סדרה חשבונית
נושא 2 מתוך 7
▼
סכום n האיברים הראשונים
Sₙ = n · (a₁ + aₙ) / 2
= n · (2a₁ + (n−1)·d) / 2
נוסחה 1 (כשיש a₁ ו-aₙ):   Sₙ = n·(a₁+aₙ)/2
נוסחה 2 (כשיש a₁ ו-d):   Sₙ = n·(2a₁+(n−1)d)/2
שתי הנוסחאות שוות — השתמש בזו שנוחה לך לפי הנתונים.
4
חשב: 1 + 2 + 3 + ... + 100
סדרה חשבונית: a₁=1, aₙ=100, d=1, n=100
S₁₀₀ = 100·(1+100)/2 = 100·101/2 = 5050
✓ S₁₀₀ = 5050
5
בסדרה: a₁ = 4, d = 3. מצא את n כך ש-Sₙ = 165.
Sₙ = n·(2·4 + (n−1)·3)/2 = 165
n·(8 + 3n − 3)/2 = 165
n·(3n + 5)/2 = 165 → n·(3n+5) = 330
3n² + 5n − 330 = 0
n = (−5 ± √(25+3960))/6 = (−5 ± √3985)/6 ≈ (−5 ± 63.1)/6
n = 58.1/6 ≈ 9.7... בדיקה: n=10: S₁₀ = 10·(8+27)/2 = 10·35/2 = 175 ≠ 165
נסה n=10: 3·100+5·10=330 → 330/2=165 ✓   n = 10
✓ n = 10
6
חשב את סכום כל המספרים הזוגיים בין 1 ל-200.
סדרה: 2, 4, 6, ..., 200   →   a₁=2, d=2, aₙ=200
n:   aₙ = 2 + (n−1)·2 = 200 → n−1 = 99 → n = 100
S₁₀₀ = 100·(2+200)/2 = 100·101 = 10100
✓ סכום המספרים הזוגיים = 10100
✖️
סדרה הנדסית (Geometric)
נושא 3 מתוך 7
▼
מה זו סדרה הנדסית?
סדרה שבה היחס בין כל שני איברים עוקבים קבוע ושווה. היחס הקבוע נסמן q.
איבר כללי — סדרה הנדסית
aₙ = a₁ · qⁿ⁻¹
q = aₙ₊₁ / aₙ (היחס בין איברים עוקבים)
אם q > 1 → סדרה עולה  |  0 < q < 1 → סדרה יורדת
q < 0 → הסדרה מחלפת סימנים (חיובי, שלילי, חיובי...)
בסדרה הנדסית: a₁ ≠ 0 ו-q ≠ 0 תמיד!
תכונה: aₙ² = aₙ₋₁ · aₙ₊₁ (האיבר האמצעי הוא ממוצע הנדסי)
7
בסדרה הנדסית: a₁ = 2, q = 3. מצא a₅ ואת n עבור aₙ = 162.
aₙ = 2 · 3ⁿ⁻¹
a₅ = 2 · 3⁴ = 2 · 81 = 162
aₙ = 162:   2 · 3ⁿ⁻¹ = 162 → 3ⁿ⁻¹ = 81 = 3⁴ → n−1 = 4 → n = 5
✓ a₅ = 162, n = 5
8
בסדרה הנדסית: a₂ = 6, a₅ = 48. מצא a₁ ואת q.
a₂ = a₁·q = 6    (I)
a₅ = a₁·q⁴ = 48    (II)
II÷I:   q³ = 48/6 = 8 → q = 2
מ-I: a₁ = 6/2 = 3
✓ a₁ = 3, q = 2
9
בסדרה הנדסית: 4, 12, 36, 108, ... מצא את האיבר הראשון הגדול מ-10000.
a₁ = 4, q = 3, aₙ = 4·3ⁿ⁻¹
4·3ⁿ⁻¹ > 10000 → 3ⁿ⁻¹ > 2500
n−1 > log₃(2500) = ln(2500)/ln(3) ≈ 7876/1.099 ≈ 7.17 → n > 8.17
n = 9: a₉ = 4·3⁸ = 4·6561 = 26244
✓ a₉ = 26244 (האיבר הראשון הגדול מ-10000)
∑
סכום סדרה הנדסית
נושא 4 מתוך 7
▼
סכום סופי (q ≠ 1)
Sₙ = a₁·(qⁿ−1)/(q−1)
= a₁·(1−qⁿ)/(1−q)
סכום אינסופי (|q| < 1)
S∞ = a₁ / (1−q)
רק כאשר −1 < q < 1
מתי קיים סכום אינסופי? רק כאשר |q| < 1 (הסדרה שואפת לאפס). אם |q| ≥ 1 — הסכום האינסופי לא מוגדר.
10
חשב: S₆ לסדרה 1, 2, 4, 8, 16, 32
a₁ = 1, q = 2, n = 6
S₆ = 1·(2⁶−1)/(2−1) = (64−1)/1 = 63
✓ S₆ = 63
11
חשב את הסכום האינסופי: 1 + 1/2 + 1/4 + 1/8 + ...
a₁ = 1, q = 1/2 (|q| < 1 → קיים S∞)
S∞ = 1 / (1 − 1/2) = 1/(1/2) = 2
✓ S∞ = 2
12
בסדרה הנדסית: a₁ = 8, S∞ = 24. מצא את q.
S∞ = a₁/(1−q) = 24 → 8/(1−q) = 24
1−q = 8/24 = 1/3 → q = 2/3
בדיקה: |2/3| < 1 ✓
✓ q = 2/3
13
בסדרה הנדסית: Sₙ = 93, a₁ = 3, q = 2. מצא n.
Sₙ = 3·(2ⁿ−1)/(2−1) = 3·(2ⁿ−1) = 93
2ⁿ−1 = 31 → 2ⁿ = 32 = 2⁵ → n = 5
✓ n = 5
🔁
סדרה כללית — נסיגה ורקורסיה
נושא 5 מתוך 7
▼
הגדרה רקורסיבית (Recurrence)
במקום לתת נוסחה ישירה, מגדירים כל איבר באמצעות האיברים הקודמים. נותנים: את האיבר/ים הראשון/ים + כלל המעבר.
14
סדרה מוגדרת: a₁ = 2, aₙ₊₁ = 3aₙ − 1. חשב a₂, a₃, a₄.
a₂ = 3·a₁ − 1 = 3·2 − 1 = 5
a₃ = 3·a₂ − 1 = 3·5 − 1 = 14
a₄ = 3·a₃ − 1 = 3·14 − 1 = 41
✓ a₂=5, a₃=14, a₄=41
15
סדרה: a₁ = 1, a₂ = 1, aₙ₊₂ = aₙ₊₁ + aₙ (סדרת פיבונאצ'י). חשב a₃–a₇.
a₃ = a₂ + a₁ = 1+1 = 2
a₄ = a₃ + a₂ = 2+1 = 3
a₅ = a₄ + a₃ = 3+2 = 5
a₆ = a₅ + a₄ = 5+3 = 8
a₇ = a₆ + a₅ = 8+5 = 13
✓ 1, 1, 2, 3, 5, 8, 13, ...

גבול סדרה

כלל השוואה לחישוב גבול
lim(aₙ/bₙ) כש-n→∞
חלק מונה ומכנה ב-nᵏ (החזקה הגבוהה ביותר) → השאר נשאף לאפס
16
מצא את גבול הסדרה: aₙ = (3n² + 5n) / (2n² − 1)
חלק הכל ב-n²: (3 + 5/n) / (2 − 1/n²)
כש-n→∞: 5/n → 0, 1/n² → 0
lim = (3 + 0)/(2 − 0) = 3/2
✓ lim aₙ = 3/2
17
מצא גבול: aₙ = (5n³ − 2n) / (n² + 7)
חלק ב-n³: (5 − 2/n²) / (1/n + 7/n³)
כש-n→∞: מונה → 5, מכנה → 0⁺
lim = 5/0⁺ = +∞ (הסדרה מתבדרת)
✓ הגבול הוא ∞ — הסדרה אינה מתכנסת
🔀
שאלות שילוב — חשבונית + הנדסית
נושא 6 מתוך 7 | הנושא הכי נפוץ בבגרות!
▼
מה זה שאלת שילוב?
נותנים קשר בין סדרה חשבונית לסדרה הנדסית — לדוגמה, שאיברים מסוימים שייכים לשתיהן, או שסכום אחת שווה לסכום השנייה.
טיפ: הגדר משתנים לכל סדרה (a₁, d לחשבונית; b₁, q להנדסית) ובנה מערכת משוואות.
18
3 איברים עוקבים יוצרים סדרה חשבונית: x−1,   2x+1,   5x−3.
מצא את x ואת האיברים.
בסדרה חשבונית: האיבר האמצעי = ממוצע השניים
2(2x+1) = (x−1) + (5x−3)
4x + 2 = 6x − 4 → 2x = 6 → x = 3
האיברים: 3−1=2, 2·3+1=7, 5·3−3=12
בדיקה: d = 7−2 = 5 = 12−7 ✓
✓ x=3, הסדרה: 2, 7, 12
19
שלושה מספרים יוצרים סדרה הנדסית: x,   x+6,   x+14.
מצא את x.
בסדרה הנדסית: (האיבר האמצעי)² = מכפלת שני הקצוות
(x+6)² = x·(x+14)
x² + 12x + 36 = x² + 14x
36 = 2x → x = 18
הסדרה: 18, 24, 32 → q = 24/18 = 4/3 ✓
✓ x = 18, הסדרה: 18, 24, 32
20
בסדרה חשבונית: a₁, a₂, a₃, ...
ידוע כי a₁, a₂, a₄ יוצרים (בסדר זה) סדרה הנדסית.
מצא את היחס q של הסדרה ההנדסית (בתלות ב-d ו-a₁).
a₁ = a₁, a₂ = a₁+d, a₄ = a₁+3d
תנאי הנדסית: (a₂)² = a₁·a₄
(a₁+d)² = a₁(a₁+3d)
a₁² + 2a₁d + d² = a₁² + 3a₁d
d² = a₁d → d(d−a₁) = 0
אם d ≠ 0: d = a₁
q = a₂/a₁ = (a₁+d)/a₁ = (a₁+a₁)/a₁ = 2
✓ q = 2 (כאשר d = a₁)
שאלת בגרות — שילוב

בסדרה חשבונית: a₁ = 2, d = 3.
בסדרה הנדסית: b₁ = 2, והאיבר השלישי שלה שווה לאיבר החמישי של הסדרה החשבונית.

א. מצא את q של הסדרה ההנדסית.
a₅ (חשבונית) = 2 + 4·3 = 14
b₃ (הנדסית) = b₁·q² = 2q² = 14 → q² = 7 → q = √7
✓ q = √7
ב. מצא את S₅ של כל אחת מהסדרות.
S₅ חשבונית = 5·(2·2 + 4·3)/2 = 5·16/2 = 40
S₅ הנדסית = 2·((√7)⁵−1)/(√7−1) = 2·(49√7−1)/(√7−1)
חישוב: (√7)⁵ = 7²·√7 = 49√7 ≈ 49·2.646 ≈ 129.6
S₅ ≈ 2·(129.6−1)/(2.646−1) ≈ 2·128.6/1.646 ≈ 156.3
✓ S₅ חשבונית = 40, S₅ הנדסית ≈ 156
🔍
אינדוקציה מתמטית
נושא 7 מתוך 7
▼
מה זו אינדוקציה מתמטית?
שיטת הוכחה לטענות שתלויות במספר טבעי n. מוכיחים שאם הטענה נכונה ל-n, היא נכונה גם ל-n+1.

3 שלבי האינדוקציה (חייבים לכתוב את כולם!)

שלב 1 — בסיס האינדוקציה
הוכח שהטענה נכונה עבור n = 1 (או הערך ההתחלתי).
שלב 2 — הנחת האינדוקציה
הניחו שהטענה נכונה עבור n מסוים (נסמן אותה ונשתמש בה).
שלב 3 — צעד האינדוקציה
הוכח שהטענה נכונה עבור n+1, תוך שימוש בהנחת האינדוקציה.
שגיאות נפוצות:
✗ לשכוח לכתוב "נניח שהטענה נכונה ל-n"
✗ לא להשתמש בהנחת האינדוקציה בשלב 3
✗ לא לסיים ב"ולכן הוכחנו ל-n+1"
21
הוכח באינדוקציה:   1 + 2 + 3 + ... + n = n(n+1)/2
שלב 1 — בסיס: n=1: צד שמאל = 1. צד ימין = 1·2/2 = 1. ✓
שלב 2 — הנחה: נניח שנכון ל-n: 1+2+...+n = n(n+1)/2
שלב 3 — צעד: נוכיח ל-n+1, כלומר: 1+2+...+n+(n+1) = (n+1)(n+2)/2
צד שמאל = [1+2+...+n] + (n+1) = n(n+1)/2 + (n+1)    ← שמנו הנחה
= (n+1)[n/2 + 1] = (n+1)·(n+2)/2 = צד ימין ✓
✓ הוכח לכל n טבעי
22
הוכח באינדוקציה:   1² + 2² + ... + n² = n(n+1)(2n+1)/6
בסיס (n=1): 1² = 1.   1·2·3/6 = 1 ✓
הנחה: 1²+2²+...+n² = n(n+1)(2n+1)/6
צעד: 1²+...+n²+(n+1)² = n(n+1)(2n+1)/6 + (n+1)²
= (n+1)[n(2n+1)/6 + (n+1)] = (n+1)[(2n²+n)/6 + (6n+6)/6]
= (n+1)·(2n²+7n+6)/6 = (n+1)(n+2)(2n+3)/6
= (n+1)·((n+1)+1)·(2(n+1)+1)/6 = צד ימין ✓
✓ הוכח לכל n טבעי
23
הוכח באינדוקציה:   3 | (n³ − n)   (3 מחלק את n³−n לכל n טבעי)
בסיס (n=1): 1³−1 = 0 = 3·0. ✓ (3 מחלק 0)
הנחה: 3 | (n³−n) — כלומר n³−n = 3k עבור מספר שלם k
צעד: נחשב (n+1)³−(n+1):
= n³+3n²+3n+1 − n−1 = (n³−n) + 3n²+3n
= (n³−n) + 3n(n+1)
= 3k + 3n(n+1) = 3[k + n(n+1)]
→ 3 מחלק את (n+1)³−(n+1) ✓
✓ הוכח לכל n טבעי
📝

a₇ = 3·2⁶ = 3·64 = 192

8. lim כש-n→∞: (4n²−1)/(2n²+n) = ?

שאלות מהבגרות האמיתית
3 שאלות מלאות מבחינות עבר | רמה מלאה
▼
שאלת בגרות 1
סדרה חשבונית | 2n+1 איברים | הוכחה + סכומים

נתונה סדרה חשבונית בת 2n+1 איברים (n הוא מספר טבעי). איברי הסדרה הם a₁, a₂, ..., a₂ₙ₊₁ והפרש הסדרה הוא d.

א. הוכח כי ההפרש בין סכום האיברים במקומות האי-זוגיים לבין סכום האיברים במקומות הזוגיים שווה לאיבר האמצעי בסדרה.
מקומות אי-זוגיים: a₁, a₃, ..., a₂ₙ₊₁ — n+1 איברים
מקומות זוגיים: a₂, a₄, ..., a₂ₙ — n איברים
Sאי-זוגי − Sזוגי = (a₁+a₃+...+a₂ₙ₊₁) − (a₂+a₄+...+a₂ₙ)
= (a₁−a₂) + (a₃−a₄) + ... + (a₂ₙ₋₁−a₂ₙ) + a₂ₙ₊₁
כל סוגריים שווה ל-−d → יש n סוגריים כאלה:
= n·(−d) + a₂ₙ₊₁ = a₂ₙ₊₁ − nd
האיבר האמצעי הוא aₙ₊₁ = a₂ₙ₊₁ − nd  ✓  (כי a₂ₙ₊₁ = aₙ₊₁ + n·d)
✓ הוכח: Sאי-זוגי − Sזוגי = aₙ₊₁

n איברים ראשונים: a₁, a₂, ..., aₙ
n איברים אחרונים: aₙ₊₂, aₙ₊₃, ..., a₂ₙ₊₁
T = Σ(aₙ₊₁₊ᵢ − aᵢ) עבור i=1,...,n

✓ T = n(n+1)d

נתונה סדרה הנדסית a₁, a₂, a₃, ... שמנתה q. הסדרה היא עולה.

מוגדרים הסכומים:
A = a₂ + a₄ + a₆ + ... + a₄₀   (איברים בעמדות זוגיות, עד מקום 40)
B = a₄ + a₈ + a₁₂ + ... + a₄₀   (כל איבר רביעי, עד מקום 40)

נתון: A/B = 10/9
א. הבע את A ואת B באמצעות a₁ ו-q.
A = a₂+a₄+...+a₄₀: סדרה הנדסית, איבר ראשון a₂=a₁q, מנה q², מספר איברים 20
A = a₁q · (q⁴⁰−1)/(q²−1)
B = a₄+a₈+...+a₄₀: סדרה הנדסית, איבר ראשון a₄=a₁q³, מנה q⁴, מספר איברים 10
B = a₁q³ · (q⁴⁰−1)/(q⁴−1)
✓ A = a₁q(q⁴⁰−1)/(q²−1)    B = a₁q³(q⁴⁰−1)/(q⁴−1)
ב. מצא את q.
A/B = [a₁q(q⁴⁰−1)/(q²−1)] ÷ [a₁q³(q⁴⁰−1)/(q⁴−1)]
= [q/(q²−1)] · [(q⁴−1)/q³] = (q⁴−1) / [q²(q²−1)]
= (q²+1)(q²−1) / [q²(q²−1)] = (q²+1)/q²
נתון A/B = 10/9:   (q²+1)/q² = 10/9
9q²+9 = 10q² → q² = 9 → q = ±3
הסדרה עולה → אם a₁>0 אז q>1, לכן q = 3
✓ q = 3
ג. בונים סדרה הנדסית חדשה: bₙ = 3·aₙ₊₁. מצא את מנת הסדרה bₙ.
bₙ = 3·aₙ₊₁ = 3·a₁·qⁿ = 3a₁·3ⁿ
bₙ₊₁/bₙ = 3a₁·3ⁿ⁺¹ / (3a₁·3ⁿ) = 3
✓ מנת bₙ = q = 3

נתונה סדרה הנדסית בת 2n איברים, מנתה q.

✦ סכום האיברים במקומות האי-זוגיים שווה פי 5 לסכום האיברים במקומות הזוגיים.
✦ בסדרה יש 3 איברים עוקבים שמכפלתם שווה ל-1.
✦ האיבר הרביעי בסדרה שווה ל-4. סכום 2 האיברים האחרונים הוא 1,280.
א. חשב את מנת הסדרה.
Sodd = a₁+a₃+...+a₂ₙ₋₁: סדרה הנדסית, מנה q², n איברים
Seven = a₂+a₄+...+a₂ₙ: סדרה הנדסית, מנה q², n איברים, איבר ראשון a₁q
Sodd/Seven = a₁/(a₁q) = 1/q
נתון: Sodd = 5·Seven → 1/q = 5 → q = 1/5
✓ q = 1/5
ב. (1) הראה כי קיים איבר בסדרה שערכו שווה ל-1.   (2) האם יתכן שהאיבר הראשון שווה ל-1?
(1) יהיו aₙ₋₁, aₙ, aₙ₊₁ שלושה איברים עוקבים: aₙ₋₁·aₙ·aₙ₊₁ = 1
בסדרה הנדסית: aₙ₋₁ = aₙ/q,   aₙ₊₁ = aₙ·q
(aₙ/q)·aₙ·(aₙ·q) = aₙ³ = 1  →  aₙ = 1 ✓
(2) אם a₁ = 1 ו-aₙ = 1, אז: aₙ = a₁·qⁿ⁻¹ = qⁿ⁻¹ = 1
→ q = 1, אבל q = 1/5 ≠ 1 — סתירה!
לכן: לא יתכן שהאיבר הראשון שווה ל-1.
✓ (1) aₙ = 1    (2) לא יתכן
ג. נתון: a₄ = 4 וסכום 2 האיברים האחרונים = 1,280. מצא את מספר האיברים בסדרה.
a₄ = a₁·q³ = a₁·(1/5)³ = a₁/125 = 4 → a₁ = 500
2 האיברים האחרונים (2n וגם 2n-1):
a₂ₙ₋₁ + a₂ₙ = a₂ₙ₋₁(1+q) = a₁·q²ⁿ⁻²·(1+1/5) = 500·(1/5)²ⁿ⁻²·(6/5) = 1280
600·(1/5)²ⁿ⁻² = 1280  →  (1/5)²ⁿ⁻² = 1280/600 = 32/15
אך 32/15 > 1 ו-(1/5)ᵏ < 1 תמיד. בדיקה: אולי q > 1 ⟹ q = 5
עם q = 5: a₁ = 4/5³ = 4/125. a₂ₙ₋₁+a₂ₙ = (4/125)·5²ⁿ⁻²·6 = 1280
(24/125)·5²ⁿ⁻² = 1280 → 5²ⁿ⁻² = 1280·125/24 = 6666.7 → לא מסתדר
💡 בבגרות המקורית: a₄ = 4 וסכום אחרונים = 1280 עם q נתון מחלק א'. בדוק את הנתונים המדויקים בשאלה שלך.
✓ שיטה: a₁ = a₄/q³, ואז a₂ₙ₋₁+a₂ₙ = a₁·q²ⁿ⁻²·(1+q) = 1280
