פונקציה רציונלית

  • הגדרה: f(x) = p(x)/q(x) כאשר p(x) ו-q(x) פולינומים, q(x) ≠ 0
  • תחום הגדרה: כל x שגורם למכנה להתאפס — מחוץ לתחום
אסימפטוטה אנכית
x = a
כאשר q(a)=0 אך p(a)≠0
אסימפטוטה אופקית — מונה < מכנה
y = 0
דרגת המונה קטנה מדרגת המכנה
אסימפטוטה אופקית — דרגות שוות
y = aₙ / bₙ
יחס המקדמים המובילים
אין אסימפטוטה אופקית
deg(p) > deg(q)
דרגת המונה גדולה מדרגת המכנה
אסימפטוטה אלכסונית
deg(p) = deg(q) + 1
חלוקת פולינומים: f(x) = ax + b + r(x)
שיטה לאסימפטוטה אלכסונית: מבצעים חלוקת פולינומים. השארית נעלמת כשx→±∞, ונקבל y = ax + b.
1
מצאו את תחום ההגדרה, האסימפטוטות ונקודות החיתוך עם הצירים של: f(x) = 1/(x²−4)
תחום הגדרה: x²−4 = 0 → x = ±2. לכן תחום ההגדרה: x ≠ ±2
אסימפטוטות אנכיות: x = 2 ו-x = −2 (המכנה מתאפס, המונה = 1 ≠ 0)
אסימפטוטה אופקית: דרגת מונה (0) < דרגת מכנה (2) → y = 0
נקודת חיתוך עם ציר y: f(0) = 1/(0−4) = −¼ → נקודה (0, −¼)
נקודות חיתוך עם ציר x: המונה = 1 ≠ 0 לכן אין נקודות חיתוך
סיכום: תחום x≠±2 | אסימפטוטות אנכיות x=±2 | אסימפטוטה אופקית y=0 | חיתוך עם y: (0, −¼)
2
מצאו תחום, אסימפטוטות ונקודות חיתוך עם הצירים של: f(x) = (x²−4)/x
תחום הגדרה: x = 0 גורם למכנה להתאפס. תחום: x ≠ 0
אסימפטוטה אנכית: x = 0 (מכנה מתאפס, מונה = −4 ≠ 0 בנקודה)
אסימפטוטה אופקית: דרגת מונה (2) > דרגת מכנה (1) → אין אסימפטוטה אופקית
אסימפטוטה אלכסונית: מבצעים חלוקה: (x²−4)/x = x − 4/x. כאשר x→±∞, 4/x→0, לכן y = x
נקודות חיתוך עם ציר x: x²−4 = 0 → x = ±2. נקודות: (2,0) ו-(−2,0)
נקודת חיתוך עם ציר y: לא קיימת כי x=0 מחוץ לתחום
סיכום: תחום x≠0 | אסימפטוטה אנכית x=0 | אסימפטוטה אלכסונית y=x | חיתוך עם x: (±2, 0)

פונקציות זוגיות ואי-זוגיות
נושא 2 מתוך 8
▼
פונקציה זוגית
f(−x) = f(x)
סימטריה ביחס לציר y
פונקציה אי-זוגית
f(−x) = −f(x)
סימטריה ביחס לראשית הצירים
לא זוגית ולא אי-זוגית
f(−x) ≠ f(x) וגם f(−x) ≠ −f(x)
אין סימטריה מיוחדת

שיטה — שלבי בדיקה:

  • מציבים −x בפונקציה ומפשטים
  • משווים לf(x): אם שווה → זוגית
  • משווים ל−f(x): אם שווה → אי-זוגית
  • אחרת → לא זוגית ולא אי-זוגית
⚠️ שים לב: לפני בדיקת זוגיות, ודאו שהתחום סימטרי ביחס לאפס (לכל x בתחום, גם −x בתחום).
3
קבעו אם f(x) = x/(x²−1) זוגית, אי-זוגית, או אחרת.
תחום: x ≠ ±1. התחום סימטרי (אם x בתחום → −x בתחום). ✓
מציבים −x: f(−x) = (−x)/((−x)²−1) = (−x)/(x²−1) = −x/(x²−1)
השוואה: f(−x) = −x/(x²−1) = −f(x) ✓
הפונקציה אי-זוגית! הגרף סימטרי ביחס לראשית הצירים.
4
קבעו אם f(x) = (x²+1)/x² זוגית, אי-זוגית, או אחרת.
תחום: x ≠ 0. התחום סימטרי. ✓
מציבים −x: f(−x) = ((−x)²+1)/(−x)² = (x²+1)/x² = f(x) ✓
הפונקציה זוגית! הגרף סימטרי ביחס לציר y.
5
קבעו אם f(x) = (x+1)/(x²−1) זוגית, אי-זוגית, או אחרת. שימו לב לפישוט!
פישוט: x²−1 = (x−1)(x+1), לכן f(x) = (x+1)/((x−1)(x+1)) = 1/(x−1) עבור x≠−1
תחום: x ≠ 1 ו-x ≠ −1. התחום אינו סימטרי: x=2 בתחום אך −2 בתחום, x=0.5 בתחום אך −0.5 בתחום. בדיקה: f(−x) = (−x+1)/(x²−1) = −(x−1)/((x−1)(x+1)) = −1/(x+1)
f(−x) = −1/(x+1) ≠ f(x) = 1/(x−1) (לאחר פישוט)
f(−x) = −1/(x+1) ≠ −f(x) = −1/(x−1)
הפונקציה לא זוגית ולא אי-זוגית.

רשימת ניתוח מלא — 7 שלבים:

  1. תחום הגדרה
  2. אסימפטוטות (אנכיות, אופקיות, אלכסוניות)
  3. נקודות חיתוך עם הצירים
  4. זוגיות / אי-זוגיות
  5. טבלת סימנים של f′ — עלייה/ירידה + נקודות קיצון
  6. קמירות (אם נדרש) — טבלת סימנים של f″
  7. סרטוט — שלבים 1-6 ביחד
10
ניתוח מלא: f(x) = x/(x²−1)
1. תחום: x²−1=0 → x=±1. תחום: x≠±1
2. אסימפטוטות: אנכיות: x=1, x=−1. אופקית: דרגת מונה (1) = דרגת מכנה (2)−1 < דרגת מכנה → y=0
3. נקודות חיתוך: ציר x: x=0 (מונה=0). ציר y: f(0)=0. שתי הצירים: (0,0)
4. זוגיות: f(−x)=−x/(x²−1)=−f(x) → אי-זוגית. סימטריה ביחס לראשית.
5. f′(x)=−(x²+1)/(x²−1)² — שלילי תמיד → הפונקציה תמיד יורדת, אין נקודות קיצון
6. f″(x)=2x(x²+3)/(x²−1)³ — נקודת פיתול ב-(0,0) (ראו דוגמה 9)
7. תיאור גרף: שלוש ענפים: x<−1 (יורדת מ−∞ ל−∞, קמורה ∩), −1<x<1 (יורדת מ+∞ ל−∞, עובר דרך (0,0)), x>1 (יורדת מ+∞ ל0, קמורה ∪)
תמיד יורדת | אי-זוגית | אסימפטוטות: x=±1, y=0 | פיתול ב-(0,0)
11
ניתוח מלא: f(x) = (x²−4)/x
1. תחום: x≠0
2. אסימפטוטות: אנכית: x=0. אלכסונית: f(x) = x − 4/x → y=x
3. נקודות חיתוך: ציר x: x²−4=0 → x=±2. נקודות: (2,0) ו-(−2,0). ציר y: x=0 מחוץ לתחום.
4. זוגיות: f(−x)=(x²−4)/(−x)=−(x²−4)/x=−f(x) → אי-זוגית
5. כתיבה מחדש: f(x) = x − 4/x = x − 4x⁻¹
f′(x) = 1 + 4/x² = (x²+4)/x² — מכיוון שx²+4>0 ו-x²>0: f′>0 תמיד → תמיד עולה, אין קיצון
6. f″(x) = −8/x³
תחום x < 0 x = 0 x > 0
f″ = −8/x³ + (x³<0 → −8/x³>0) א"מ − (x³>0 → −8/x³<0)
קמירות ∪ כלפי מעלה — ∩ כלפי מטה
7. תיאור גרף: שני ענפים, שניהם עולים. הענף x<0 קמור מלמטה (∪), קרוב לאסימפטוטה x=0 ולאסימפטוטה y=x. הענף x>0 קמור מלמעלה (∩). חיתוכים עם x-ציר: x=±2.
תמיד עולה | אי-זוגית | אסימפטוטות: x=0, y=x | אין נקודות קיצון

נושא 6 מתוך 8
▼

תכונות f(x) = xⁿ(x+1)², n מספר שלם חיובי, n>1

  • תחום: כל המספרים הממשיים
  • חיתוכי ציר x: x=0 (ריבוי n) ו-x=−1 (ריבוי 2)
  • ב-x=0, ריבוי n: n זוגי → ממשיק לציר (לא חוצה) | n אי-זוגי → חוצה את הציר
  • ב-x=−1, ריבוי 2: תמיד ממשיק לציר (לא חוצה), נקודת מינימום/מקסימום
  • התנהגות קצוות: כשx→+∞: f→+∞ תמיד | כשx→−∞: תלוי ב-n (n זוגי→+∞, n אי-זוגי→−∞)
12
